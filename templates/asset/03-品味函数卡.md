# 03 · 品味函数卡

> 模板类型：原力资产 / 原力母体部件 02  
> 正典位置：`原力母体 = 注意力雷达 × 品味函数 × 身体化能力 × 生命经验 × 长期承诺`  
> 使用目标：识别我如何判断什么值得做、什么不值得做。

---

## 1. 用途

品味函数用于回答：

```text
我如何判断什么值得做、什么不值得做？
```

品味不是审美偏好。

品味是价值排序系统。

在 AI 时代，执行越来越便宜，生成越来越容易，真正稀缺的是：

```text
谁知道什么值得生成。
```

---

## 2. 适用场景

```text
1. 内容、产品、战略、创业方向太多，无法判断取舍时。
2. 感觉自己有审美和判断，但还没形成清晰标准时。
3. 需要训练 AI 识别“什么像我、什么不像我”时。
4. 团队需要统一判断标准时。
5. 想把个人品味变成可复用的产品 / 内容 / 方法论标准时。
```

---

## 3. 输入材料

```text
我真正欣赏的作品 / 公司 / 人 / 产品：
我强烈不认同的作品 / 公司 / 人 / 产品：
我过去做过的高质量选择：
我过去后悔的低质量选择：
我愿意为之牺牲短期效率的标准：
我绝不愿意降低的底线：
```


### 4.2 我认为什么不值得做

```text
不值得做的事情通常具备：
1.
2.
3.
4.
5.
```

### 4.3 我的高级 / 伪高级判断

```text
我认为的高级是：

我认为的伪高级是：

我如何区分高级和伪高级：
```

### 4.4 我的真问题 / 伪需求判断

```text
我认为的真问题是：

我认为的伪需求是：

我如何判断一个问题是否值得下注：
```

### 4.5 我愿意坚持的标准

```text
我宁可慢一点，也不愿降低的标准：

我宁可少赚一点，也不愿降低的标准：

我宁可不做，也不愿违背的标准：
```

### 4.6 品味函数一句话

```text
我的品味函数，是我在 ______ 场景中，用 ______ 标准判断 ______ 是否值得长期投入的价值排序系统。
```

---

## 5. 判断标准

一个高质量品味函数，应满足：

```text
1. 能判断取舍，而不是只表达偏好。
2. 能解释为什么高级，而不是只说喜欢。
3. 能稳定用于不同场景，而不是一次性判断。
4. 能帮助 AI 过滤普通答案。
5. 能转化为内容、产品、服务、品牌或组织标准。
```

评分：

```text
A：我有清晰标准，能稳定判断，并能训练别人 / AI 使用。
B：我有强烈偏好，但标准还需要语言化。
C：我只是喜欢 / 不喜欢，无法解释原因。
D：我的判断主要来自流行、权威或短期情绪。
```
