---

## 0. 使用场景

当你不知道自己的原力母体在哪里时，不要先问“我喜欢什么”，而是先问：

```text
我过去一周的活里，哪些其实只是信息处理？
哪些真的来自不可复制的我？
```

本卡用于完成 A1 的第一个动作：

```text
同质化能力清算 → 算力活 / 原力活分堆 → 原力母体候选线索
```

## 3. 第二步：判断是否为算力活

```text
这件事的主要价值是否来自信息处理？

这件事是否主要依赖整理、归纳、套模板、检索、格式化、汇总？
```

## 4. 第三步：判断是否为原力活

原力活的判断标准：

```text
这件事是否依赖我的独特判断？
这件事是否来自我的长期注意力、品味、经验、关系或生命主题？
别人即使照着做，也很难有同样效果吗？
我做这件事是否更有能量、更清楚、更像我？
它是否持续生成新的判断、作品、关系或意义？
```

## 5. 第四步：计算原力活占比

```text
原力活占比 = 原力活耗时 / 总工作耗时
```

| 项目 | 数值 |
|---|---:|
| 总工作时间 |  |
| 算力活时间 |  |
| 原力活时间 |  |
| 混合任务时间 |  |
| 原力活占比 |  |

### 解释

```text
低于 20%：你大部分生命正在消耗在机器更擅长的任务上，套利空间巨大。
20%-40%：你已经开始碰到母体，但还没有重组工作系统。
40%-60%：你有明显原力工作区，可以进入 A2 回到母体。
60% 以上：你已具备较强原力配置，下一步是 A3 外化与 A4 显化。
```
