# 原力资产 · 概念体系

> 找到你：把“你”这颗最贵算法，借助 AI 放大成会复利的资产。

---

## 1. 一句话定义

**原力资产，是被 AI 放大的“你”。**

```text
原力资产 = 「你」 × AI
```

「你」不是简历、经验、技能点，而是不可复制的深度结构：非对称优势、非理性痴迷、非线性回报、原型、阴影、天才区、意义主权。

AI 不是普通工具，而是最强算力与镜像放大器。

---

## 2. 核心方法论：原力U

```text
觉察 → 高能 → 阴影 → 整合 → 资产 → 复利
```

它不是平铺清单，而是一条 U 型下潜线：

```text
从症状进入结构
从结构潜到阴影源头
从源头整合上升
最终浮出为资产
```

---

## 3. 五阶结构

### 阶 1：觉察

问题：我是否承认“你”才是最贵算法？

关键词：

```text
算力鸿沟 / 你是最贵算法 / 路径反转 / 元AI / 双螺旋
```

### 阶 2：高能

问题：我的高能结构在哪里？

关键词：

```text
三圈交集 / 12 原型 / 天才区 / 非对称优势 / 非理性痴迷 / 非线性回报
```

### 阶 3：阴影

问题：我最不愿意看的部分，是否藏着最大能量？

关键词：

```text
金色阴影 / 大象骑象人 / 高敏感超能力 / 阴影炼金术
```

### 阶 4：整合

问题：我能否用 AI 照见自己、整合自己、共生自己？

关键词：

```text
上下文工程 / 我的说明书 V1-V3 / 元AI / 镜子 / 放大器 / 预言家
```

### 阶 5：资产

问题：我是否按下确定键，形成可被市场认领的母体？

关键词：

```text
母体定义卡 / 天选用户 / 确定键 / 原力资产显形
```

### 出口：复利

问题：这个资产是否守得住、滚得大、穿越周期？

关键词：

```text
守富 > 创富 / 30 法则 / 数据飞轮 / 留存复利 / 原力人生融合
```
