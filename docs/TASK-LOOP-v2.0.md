# Whiteboard Task Loop v2.0

Goal: turn a selected canon node into an executable loop.

Loop:

1. Select node.
2. Read Agent Contract.
3. Draft task.
4. Run dry-run.
5. Request approval when needed.
6. Execute within boundary.
7. Capture evidence.
8. Write feedback note.
9. Write evolution note.
10. Backwrite to target.

Every task needs objective, context, steps, acceptance, evidence, risk note, and writeback target.
