OS 不是工具箱，而是一个懂你、记得住、会判断、能动手、每次变强的工作生命体。

Soul

文脉

灵魂：我是谁、为什么做、边界是什么。

Memory

大脑

记忆：知识、案例、经验、判断沉淀。

Judgment

地图

判断：因果图、战略图、关键战役、决策结构。

Action

链路

手脚：OSA、agent、workflow、自动化执行。

Constitution

递归进化 / 技能统帅 / 人类友好。

Method

OSA + 信息五层
