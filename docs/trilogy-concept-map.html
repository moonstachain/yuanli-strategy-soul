D7 · state machine
Part I · 原力资产

你还没有稳定母体时，先别急着创业。

重点是发现母体、回到母体、获得原力、显化原力，最终生成创业输入包。

进入 Asset Tool
Part II · 原力创业

你已有验证线索时，判断市场如何认得你。

重点是借势、品类、模式、壁垒，最终生成 OS 回写包。

进入 Startup Tool
Part III · 原力OS

你已经在交付时，必须解决组织不失忆。

重点是文脉、大脑、战略地图、行动链路，让系统递归进化。
