第一部：原力资产 · 七阶资产生成机制

原力母体是核心，但不是全部。《原力资产》完整机制是：先清算同质化能力，再返源母体、拆解成形、AI外化、市场验证、OS回写，最后形成可进化身份。

Formula

主公式

原力资产 = 原力母体 × AI外化 × 市场验证 × OS回写

Matrix

原力母体

注意力雷达 × 品味函数 × 身体化能力 × 生命经验 × 长期承诺。

Result

终态

不是固定身份，而是会被外化、验证、回写、复利的原力母体。

Entry

交互页

六张核心卡可填写、评分、导出。

进入 asset.html

七阶总览

清算停止把同质化能力误认为资产。
返源回到持续生成差异的原力母体。
成形把母体拆成五个部件。
外化让 AI 把母体变成可见资产。
验证确认真实世界是否需要。
回写把反馈更新回母体与OS。
复利母体越用越清楚、越值钱。

七阶细化：输入 / 判断 / 输出 / 接口

阶定义子模块输入判断标准输出接口
清算识别 AI 正在同质化的能力。可学习 / 可表达 / 可执行 / 可模仿 / 可流程化。工作清单、优势清单、AI替代任务。AI+模板能否接近？越普及是否越便宜？同质化能力清单、自动化清单。进入返源。
返源从技能和标签背后找到生成差异的源头。长期注意力、重复判断、低级答案厌恶、他人获得物、母题浮现。高低能事件、被求助问题、长期主题。是否长期稳定、持续生成差异、解释多个选择？母体候选、母体一句话。进入成形。
成形把母体拆成可操作五部件。注意力雷达、品味函数、身体化能力、生命经验、长期承诺。母体候选、品味样本、能力案例、生命经历。五部件是否互相支撑？经验是否指向承诺？六张资产卡。进入外化。
外化用 AI 把母体变成可见资产。语言、模型、产品、流程、文脉外化。六张资产卡、案例、输出样本、用户反馈。AI 是否放大母体而非平均答案？母体说明书、方法论、产品原型、OS文脉种子。进入验证。
验证确认母体是否被世界需要。靠近、相信、付费、复用、行动、共建验证。外化产物、目标人群、产品原型、互动记录。是否有真实行为？是否出现天选用户和贵问题？天选用户、贵问题、心理账户、市场验证记录。交给原力创业。
回写把反馈写回母体与OS。市场、身体、关系、任务、AI、OS回写。市场反馈、交付复盘、身体状态、AI偏差。反馈是否结构化？是否改变下一次行动？母体更新、OS回写、新判断标准。进入复利。
复利多轮外化验证回写后形成可进化身份。母体复利、AI复利、市场复利、OS复利、财富接口。多轮验证、版本记录、反馈曲线、回写日志。下一次是否更容易、更准、更可复用？会复利的原力母体、创业输入包、OS文脉包。进入原力创业或下一轮升级。

怎么守住、不被拿走？

四关细化：输入 / 判断 / 输出 / 接口

关定义子模块输入判断标准输出接口
原力借势判断这个时代为什么现在需要我的母体。宏周期势、中结构势、微入口势、合力匹配、时间窗口。母体输入包、市场验证、趋势资料、用户变化、政策技术渠道变化。是否只是风口？是否有结构变化？是否和母体匹配？势能地图、机会假设、入场窗口、优先级。进入品类独创。
品类独创把母体翻译成市场能识别、记住、付费的新入口。甜用户、贵任务、心理账户、旧品类对标、新品类命名、入脑表达。势能地图、天选用户、贵问题、用户语言、旧方案。用户是否一听就懂？能否转述？能否区分旧方案？新品类定义、价值主张、心理账户定位、命名候选。进入模式升维。
模式升维把一次性交付升级为可复制、可复利、可扩张的利润容器。前链路、后链路、财链路、产品阶梯、数据反馈。品类定义、用户、价值主张、交付能力、价格假设、渠道内容资产。增长是否人肉崩溃？是否有毛利现金流？是否可复购？利润容器、产品阶梯、三链路画布、交付SOP、增长指标。进入壁垒锁定。
壁垒锁定让跑通的生意不被轻易复制、拿走或替代。虚壁垒、实壁垒、入壁垒、出壁垒、飞轮壁垒。利润容器、案例、数据资产、品牌、渠道、交付系统。别人复制要多久？用户离开成本是否上升？系统是否越跑越强？财富飞轮、壁垒地图、16控制点、资产护城河、OS回写包。交给原力OS。

原力借势四层

宏周期势

康波、技术范式、资本周期、AI渗透率。先判断大时间。

输出K5存量 / K6增量 / 千日窗口。

中结构势

政策、经济、社会、技术、产业链。判断结构是否正在变化。

输出PEST / 产业价值链变化。

微入口势

用户心智、心理账户、渠道窗口、预算迁移。判断入口是否存在。

输出入口场景与预算账户。

合力匹配

我的母体是否刚好适配这股势，不追风口。

输出机会优先级。

第三部：原力OS · 四组件与递归回写机制

OS 不是工具箱，而是把文脉、知识、判断、行动链路装配成会递归回写、持续进化的工作生命体。

Soul

文脉

我是谁、为什么做、边界是什么。

Memory

大脑

我知道什么、有哪些案例和知识。

Judgment

地图

我如何决策、取舍、看因果。

Action

链路

我如何行动、交付、自动化。

Loop

递归回写

每次任务如何让下一次更强。

OS细化：输入 / 判断 / 输出 / 接口

模块定义子模块输入判断标准输出接口
文脉OS 的灵魂，回答我是谁。母体说明书、长期愿景、品味函数、禁止事项、表达风格。母体总卡、品味卡、承诺卡、表达样本、边界。AI是否能稳定理解我？是否知道不该做什么？文脉卡、我的说明书、AI使用协议。交给大脑。
大脑OS 的记忆，回答我知道什么。知识库、案例库、术语库、样例库、数据库。书稿、源稿、项目案例、反馈、模板填写稿、数据。是否可检索、可信、有版本、可调用？大脑库卡、知识索引、案例索引、术语表。交给地图。
地图OS 的判断系统，回答我怎么决策。因果地图、战略地图、用户地图、商业地图、决策地图。大脑资料、市场验证、创业四关结果、任务复盘、战略目标。是否能帮助取舍、解释因果、指导行动、被更新？决策地图、变量因果图、机会地图、行动阈值。交给链路。
链路OS 的手脚，回答我怎么行动。OSA任务链、SOP链、Agent链、内容链、交付链。决策地图、任务目标、工具能力、模板、验收标准。能否重复执行？能否交给人或AI？是否有验收？链路卡、SOP、Agent workflow、Codex任务。进入回写。
递归回写让每一次任务增强四组件和母体。Evolution Note、信息五层、Skill蒸馏、版本升级、母体回写。任务输出、复盘、反馈、失败记录、成功模式、工具记录。是否产生新模式？能否蒸馏skill？是否明确回写位置？Evolution Note、Skill卡、四组件更新、母体升级记录。回到资产与创业。
