第一部：原力资产四步

显化原力只是市场验证与创业接口，不等于完整变现；完整商业化交给第二部。

A1

发现母体

从同质化能力、外部标签和二手共识中，识别持续生成差异的底层生命结构。

不是兴趣、人格标签、职业优势、人设包装。问题AI清算之后，我身上还剩什么会持续生成差异？原子同质化清算、异常注意、高能事件、低级答案厌恶、他人获得物、母题浮现。输出母体候选、母题清单、母体一句话假设。回写原力母体总卡 / OS文脉。
A2

回到母体

停止被外部评价、短期机会和工具兴奋牵引，以母体重组选择与边界。

不是退回舒适区、自我感动、排斥市场。问题如果以母体为中心，哪些事保留，哪些事停止？原子注意力雷达、品味函数、身体化能力、生命经验、长期承诺、边界弃绝。输出母体定义、五部件卡组、不再做清单。回写asset.html / OS文脉卡 / 个人战略边界。
A3

获得原力

让母体被训练、AI外化、作品化、关系化，形成可调用高能能力。

不是突然开悟、神秘能量、学更多AI工具。问题母体如何变成判断力、表达力、创造力、连接力和行动力？原子AI镜像、概念外化、模型外化、作品外化、练习闭环、原力指标。输出原力能力包、母体说明书、方法论模型、作品样本。回写OS大脑样例库 / 方法论资产库。
A4

显化原力

把原力放进真实世界，验证是否被人需要、相信、复用、付费和共建。

不是完整商业模式、规模化变现、创业全部。问题这个原力是否真的被世界需要？原子天选用户、贵问题、心理账户线索、最小显化、市场六问、创业输入包。输出市场验证记录、创业输入包、母体回写记录。回写机会假设库 / 原力借势卡。

第二部：原力创业四步

原力创业把已经显化的原力放入时代势能、用户心智、商业模式与系统壁垒。

B1

原力借势

判断这个时代为什么现在需要我的原力，并找到最适合进入市场的势能窗口。

不是追风口、蹭热点、宏观叙事自嗨。问题这个时代为什么现在需要我？原子宏周期势、中结构势、微入口势、合力匹配、时间窗口、势能排序。输出势能地图、机会假设、入场窗口。回写OS机会地图 / 长期承诺。
B2

品类独创

把原力翻译成市场能识别、能记住、愿意付费、愿意转述的新入口。

不是起好听名字、包装卖点、只做差异化话术。问题市场如何认得我？原子甜用户、贵任务、心理账户、旧品类对标、新品类命名、入脑表达。输出新品类定义、一句话价值主张、心理账户定位。回写文脉词典 / 用户语言库。
B3

模式升维

把被市场认出的原力品类，升级为可复制、可复利、可扩张的利润容器。

不是卖更多时间、简单涨价、堆产品、只追流量。问题如何持续赚钱、可复制？原子前链路、后链路、财链路、产品阶梯、数据反馈、人机分工。输出利润容器、产品阶梯、三链路画布、交付SOP。回写OS链路 / 客户成功库 / 数据指标库。
B4

壁垒锁定

让跑通的利润容器不被轻易复制、拿走、替代，并越跑越强。

不是自我安慰式护城河、保密、先发幻觉。问题如何守住、不被拿走？原子虚壁垒、实壁垒、入壁垒、出壁垒、飞轮壁垒、母体壁垒。输出财富飞轮、壁垒地图、16控制点、OS回写包。回写OS地图 / 壁垒库 / 母体升级。

第三部：原力OS四步

递归回写不是第五步，而是贯穿“一纸文脉、一个大脑、一张地图、一条链路”的底层机制。

C1

一纸文脉

把我是谁、为什么做、原力母体、长期边界和表达风格写成最小正典文件。

不是个人简介、品牌文案、愿景口号、提示词集合。问题AI 如何真正懂我？原子母体说明、长期承诺、品味函数、战略边界、语言风格、禁止事项。输出一纸文脉、我的说明书V3、AI使用协议。回写CONSTITUTION / OS文脉卡 / 元AI上下文。
C2

一个大脑

把知识、案例、术语、样例、数据和经验沉淀成可检索、可调用、可更新的记忆系统。

不是文件夹堆积、资料收藏、知识库幻觉。问题系统应该记住什么，才能让下一次不从零开始？原子知识库、案例库、术语库、样例库、数据库、索引层。输出一个大脑、知识索引、案例索引、术语表。回写source/indexes / extracts / 案例库。
C3

一张地图

把文脉和大脑组织成能判断方向、取舍、优先级和路径的决策结构。

不是思维导图、战略口号、漂亮但不能决策的图。问题系统如何知道什么更重要、先做什么、怎么判断？原子目标地图、用户地图、机会地图、因果地图、商业地图、决策阈值。输出一张地图、决策地图卡、变量因果图、战略路径图。回写OS地图 / 项目看板 / Codex任务树。
C4

一条链路

把地图中的判断转化为可重复执行、可交给人或AI、可验收、可回写的行动流程。

不是待办事项、临时任务、一次性交付、无验收流程。问题如何把判断变成可复用、可执行、可自动化的行动？原子OSA任务链、SOP链、Agent链、Codex链、内容链、交付链、回写链。输出一条链路、SOP、Agent workflow、Codex-ready task、Skill蒸馏卡。回写codex / project / templates / skills / 母体升级。
