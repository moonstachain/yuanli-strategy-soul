## 1. 一句话定义

**原力创业，是拿着已经确认的原力资产，去寻找它最该搭上的时代顺风，并把它变成可通关、可复利、守得住的生意。**

核心公式：

```text
财富 = 借势 × 合力
```

```text
合力 = 原力资产已经生成的内部不变量
借势 = 原力创业首先要完成的外部势能判断
```

二者是乘法关系，任一为 0，整体归零。

## 2. 核心结构：新四关

```text
原力借势 → 品类独创 → 模式升维 → 壁垒锁定
```

四关不是四个模块，而是严格串行的通关链。

## 4. 四关内部结构

### 第一关：原力借势

```text
原力借势 = 宏周期势 × 中结构势 × 微入口势 × 合力匹配
```

目标：拿着已经确认的原力资产，去寻找它最该搭上的时代顺风。

| 层级 | 看什么 | 工具 |
|---|---|---|
| 宏周期势 | 康波、技术范式、资本周期、AI渗透率 | 康波图、技术S曲线 |
| 中结构势 | 政策、经济、社会、技术、产业价值链 | PEST / PESTEL / 产业链图 |
| 微入口势 | 用户心智、心理账户、渠道窗口、预算迁移 | JTBD、心理账户、甜用户 |
| 合力匹配 | 原力资产是否刚好适配这股势 | 三圈交集、母体定义卡 |

通关检查：

```text
1. 能 30 秒说清：我借的是哪一股势。
2. 能用康波 / 技术 / 产业 / PEST 至少三类证据证明这股势存在。
3. 能说明：为什么是现在，不是三年前，也不是三年后。
4. 能证明：我的原力资产和这股势同向，而不是硬蹭风口。
5. 能输出明确进入点：甜用户、贵任务、预算来源、品类机会。
```

### 第二关：品类独创

```text
三角定位 = 甜用户 × 贵任务 × 入脑表达
```

目标：进入用户的高阶心理账户，而不是在功能账户里卷价格。

### 第三关：模式升维

```text
三链贯通 = 前链路 × 后链路 × 财链路
```

目标：把单点价值变成利润容器。

### 第四关：壁垒锁定

```text
四维护城 = 虚 × 实 × 入 × 出
```

目标：让利润守得住、飞轮转得起、资产可复利。

## 5. 四获得物

```text
势能地图 → 心理账户 → 利润容器 → 财富飞轮
```

## 6. 与原力资产的接口

原力创业接收原力资产的输出：

```text
母体定义卡
天才区
原型力
非理性痴迷
非对称优势
非线性回报
天选用户线索
```

然后先完成借势判断：

```text
康波势
技术势
产业势
政策 / 经济 / 社会 / 技术结构
用户心智窗口
预算迁移窗口
合力匹配度
```

再把它市场化为：

```text
品类
用户
任务
表达
模式
壁垒
```

## 7. 与原力OS的接口

原力创业完成后，必须把经验交给 OS：

```text
哪些动作可以复用？
哪些知识可以沉淀？
哪些判断可以模型化？
哪些交付可以链路化？
哪些经验可以蒸馏成 skill？
```

否则创业只是一次性项目，不是 ʸx 生长。
