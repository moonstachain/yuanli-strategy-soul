原力OS工具
写成系统，递归回写

这是原力战略三部曲的第三部交互工具。它承接 Startup Tool 的 OS 回写包，帮助你搭建：一纸文脉、一个大脑、一张地图、一条链路。特别校准：这里的“一张地图”是战略地图 / 能力组件地图 / 事业系统地图，不是普通思维导图。

原力OS四步：一纸文脉 → 一个大脑 → 一张地图 → 一条链路 一张地图 = 战略地图 / 能力组件地图 / 事业系统地图 递归回写不是第五步，而是贯穿四步的底层机制。
0%当前完成度。本地自动保存，不上传服务器；先补文脉和地图，再谈自动化链路。
未评分成熟度把系统翻译成状态机：待搭建、OS雏形、可运行OS、可递归OS。
输出回写导出 CONSTITUTION、大脑索引、战略地图、Codex-ready 链路与模板/skill 沉淀。

四步系统路径

01 一纸文脉把我是谁、为什么做、边界与风格写成最小正典。
02 一个大脑把知识、案例、术语、样例、数据沉淀为记忆系统。
03 一张地图战略地图 / 能力组件地图，决定方向、取舍、优先级与路径。
04 一条链路把战略判断变成可重复执行、可验收、可回写的行动流程。
