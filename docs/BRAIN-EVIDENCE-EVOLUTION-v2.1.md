# Brain / Evidence / Evolution v2.1

Goal: every task leaves reusable memory.

Record types:

- Evidence: source, output, decision, proof.
- Feedback Note: what happened, gap, next action.
- Evolution Note: input, output, acceptance, writeback, next reuse.

Default writeback locations:

- brain/evidence
- brain/records
- brain/evolution-notes
- docs
- project

Rule: no output is complete until evidence and writeback target are named.
