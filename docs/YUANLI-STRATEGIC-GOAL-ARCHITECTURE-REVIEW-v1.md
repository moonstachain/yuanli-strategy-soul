Date: 2026-06-30
Stage: Pre-External Validation OS

## Strategic Goal

原力战略三部曲的战略目标不是单纯做课程、书稿或知识库，而是：

```text
把创始人的不可复制性，转化为可复利、可验证、可回写、可传承的事业操作系统。
```

它的长期形态是：

```text
个人原力发现系统 × 创业商业化系统 × AI时代事业OS
```

## Mother Axiom

最高判断标准仍然是：

```text
ʸx：做会自我繁殖的事，不做一次性的事。
```

因此，任何资产都要接受五个问题检验：

```text
能否执行？
能否验证？
能否回写？
能否复用？
能否让下一轮更强？
```

## Architecture

当前系统已经形成五层：

```text
L1 Canon Layer      正典层
L2 Asset Layer      内容资产层
L3 Brain Layer      一个大脑层
L4 PMO Layer        项目治理层
L5 Validation Layer 外部验证层
```

### L1 Canon Layer

核心是 `CONSTITUTION.md` 和 A1-C4 十二个正典模块。

### L2 Asset Layer

核心是 `assets/` 下 12 个模块的 seed 内容资产包。

### L3 Brain Layer

核心是 C2 一个大脑、`brain/INDEX.md`、`brain/SCHEMA.md`、`brain/QUERY-EXAMPLES.md`、`brain/WRITEBACK-LOG.md`。

### L4 PMO Layer

核心是 `docs/pmo-dashboard-data.js`、PMO 页面、同步脚本和 GitHub Issues。

### L5 Validation Layer

当前核心是 B2 验证包与 #130 小范围验证工作坊。
