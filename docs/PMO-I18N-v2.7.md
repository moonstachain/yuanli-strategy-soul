# PMO I18N v2.7

## Goal

Make PMO pages understandable for Chinese learners while keeping English terms for GitHub / engineering consistency.

## Rule

Use Chinese-first bilingual labels:

```text
中文主标题 / English subtitle
```

## Applied pages

- `docs/pmo-v27.html`

## Entry redirects

- `pmo.html` -> `docs/pmo-v27.html`
- `docs/pmo.html` -> `pmo-v27.html`

## Copy style

- Main title: Chinese + English
- Section heading: Chinese first, English second
- KPI label: Chinese first, English second
- Technical terms: keep original English after Chinese explanation

## Reason

Chinese learners need fast comprehension. English terms remain visible so they can map PMO concepts to GitHub, Issues, Evidence, Evolution Notes, and engineering workflow vocabulary.
