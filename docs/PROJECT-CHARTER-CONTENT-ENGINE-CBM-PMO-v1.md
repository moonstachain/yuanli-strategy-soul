# Project Charter · Yuanli Content Engine CBM-PMO Program v1

Date: 2026-06-30  
Epic: #183  
Status: draft / formal charter / not merged

## 1. North Star

Turn the Yuanli canon into a repeatable content production, distribution, feedback, evidence, and writeback system.

```text
Canon Node → Topic Map → Channel Adaptation → Production → Publish/Test → Feedback → Evidence → Evolution → Backwrite
```

## 2. Why This Program Exists

The content workstream should not continue as scattered chat-driven tasks. It must become a governed production program where every content artifact can be traced to canon, evidence, channel, feedback, and reuse path.

## 3. Program Type

Production program / growth capability program.

## 11. Guardrails

```text
No feedback, no Done.
No evidence, no green.
No sensitive source exposure.
No external publish from private source without human approval.
No automatic merge.
```
