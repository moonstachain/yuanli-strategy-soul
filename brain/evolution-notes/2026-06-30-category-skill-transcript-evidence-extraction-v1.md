## Strategic Reason

The current `USER-LANGUAGE-TEST-v1.md` needs better evidence before interviews. `llm-wiki` transcripts are not final user-test evidence, but they are high-value artifact material for:

```text

3. extracting expensive problem language
4. surfacing budget-pull reasons
5. identifying objections to test
```

## Source Tier Rule

```text
T0 = target-user user-language test transcript
T1 = founder / client / learner interview transcript
T2 = course / private board / work meeting transcript
T3 = digest / summary / case writeup
```

Only T0 can support user-tested status. T1/T2/T3 can support extraction and test design, but not validation upgrade by themselves.
