## 3. Evidence of Governance Decision

The core governance decision is:

```text
Repo activity is not value.
Repo value = CBM Cell + Evidence + Next C3/C4 + Writeback Target + Reuse Path.
```

This prevents the GitHub portfolio from becoming a noisy repo inventory and converts it into a strategic capability portfolio.
