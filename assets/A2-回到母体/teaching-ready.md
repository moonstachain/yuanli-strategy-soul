# A2 · 回到母体 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能用母体重组注意力、品味、能力、经验和边界，形成“保留 / 停止 / 加强”的选择系统。

## 30-60 min Teaching Flow

1. 从机会牵引回到母体牵引。
2. 建立注意力雷达：什么会天然抓住你？
3. 建立品味函数：你为什么觉得某些答案低级？
4. 建立不再做清单：哪些机会消耗母体？
5. 母体总卡：把注意力、品味、身体化能力、生命经验、长期承诺合并。

## Correct Case

创始人停止追逐短期热门项目，把精力集中到自己最能持续生成判断的复杂行业翻译与战略秩序上。

## Anti-pattern

把“回到母体”理解成退回舒适区、拒绝市场、自我感动。

## Assignment

完成：

```text
原力母体总卡 + 注意力雷达卡 + 不再做清单
```

## Script

“回到母体不是不进入世界，而是用自己的源头重新决定进入世界的方式。”

## Evidence Boundary

当前为 teaching-ready。需要真实学员能用卡片做出 3 个停止事项和 3 个保留事项后进入 validated teaching。
