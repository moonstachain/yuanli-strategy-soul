# A2 · 回到母体 · Teaching Outline

PMO Issue: #87
Asset Type: teaching-outline

## 一句话目标

帮助学员从外部评价、身份噪音和短期机会中回收注意力，重新回到自己的原力母体。

## 适用对象

- 已经识别出初步母体，但容易被机会牵走的人
- 创业过程中频繁换方向的人
- 专家型个体转型期的人
- 被流量、收入、身份评价影响判断的人

## 学完以后，学员应能完成

1. 识别自己偏离母体的典型信号。
2. 区分“机会吸引”和“母体牵引”。
3. 写出自己的不做清单。
4. 建立回到母体的日常校准机制。
5. 形成 A2 回到母体校准卡。

## 课程结构

### 1. 为什么找到母体后仍会偏离

- 外部机会太多
- 身份奖励太强
- 短期现金流焦虑
- 模仿成功者路径
- 把市场反馈误当长期方向

### 2. 偏离母体的五个信号

```text
能量下降 / 判断变钝 / 表达失真 / 关系消耗 / 无法复用
```

### 3. 机会吸引 vs 母体牵引

第一性问题：这个机会是在放大我的母体，还是在消耗我的母体？

### 4. 不做清单

第一性问题：哪些事虽然有收益，但会让我离母体更远？

### 5. 回到母体的三层校准

```text
日校准：今天是否偏离？
周校准：本周是否复用？
季校准：方向是否仍然服务长期承诺？
```

## 输出作业

```text
A2 回到母体校准卡
```

字段：

- 我的偏离信号
- 我的机会判断标准
- 我的不做清单
- 我的回归动作
- 我的周复盘问题

## 验收标准

- 能识别偏离
- 能拒绝错误机会
- 能把注意力回收到母体
- 能进入 A3“获得原力”
