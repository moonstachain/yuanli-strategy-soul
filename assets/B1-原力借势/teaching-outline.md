# B1 · 原力借势 · Teaching Outline

PMO Issue: #90
Asset Type: teaching-outline

## 一句话目标

帮助学员把个人原力放入更大的时代趋势、技术周期、市场结构和文化叙事中，获得外部势能。

## 适用对象

- 已完成 A1-A4 原力资产识别与显化的人
- 准备从个人能力进入创业场景的人
- 有产品或服务雏形，但缺少趋势判断的人
- 想避免只靠个人努力硬推的人

## 学完以后，学员应能完成

1. 识别与自己原力相关的外部势能。
2. 区分风口、趋势和长期结构变化。
3. 找到自己的借势入口。
4. 设计一个小规模借势实验。
5. 写出 B1 原力借势卡。

## 课程结构

### 1. 为什么创业不能只靠原力本身

个人原力是内在源头，但创业需要外部势能。没有借势，容易变成高能低效。

### 2. 四类可借之势

```text
技术势 / 市场势 / 文化势 / 关系势
```

### 3. 风口 vs 趋势 vs 结构变化

第一性问题：这个变化会不会持续改变用户行为、成本结构或分发方式？

### 4. 原力与外部势能的匹配

第一性问题：哪个外部趋势正在放大我的原力，而不是消耗我的原力？

### 5. 最小借势实验

```text
一个趋势假设 + 一个用户场景 + 一个最小产品或内容 + 一个反馈指标
```

## 输出作业

```text
B1 原力借势卡
```

字段：

- 我的原力资产
- 可借之势
- 趋势判断
- 用户变化
- 借势入口
- 最小实验
- 反馈指标

## 验收标准

- 能说清趋势，不只说风口
- 能解释趋势如何放大原力
- 有最小实验
- 有反馈指标
- 能进入 B2“品类独创”
