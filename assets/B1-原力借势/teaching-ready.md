# B1 · 原力借势 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能判断“这个时代为什么现在需要我”，并形成宏周期势、中结构势、微入口势与合力匹配的势能地图。

## 30-60 min Teaching Flow

1. 借势不是追热点。
2. 宏周期势：K5 存量与 K6 增量。
3. 中结构势：PEST、产业链、价值迁移。
4. 微入口势：用户心智与预算迁移。
5. 合力匹配：我的原力是否刚好适配。
6. 入场窗口：早、刚好、晚。

## Correct Case

留学生回国增长、预算提高、AI 可现场呈现方案，和高客单婚礼策展能力合力，形成顺势增长。

## Anti-pattern

用宏观大词解释小生意，或看到热点就冲。

## Assignment

完成：

```text
势能地图 + 入场窗口判断 + 机会假设卡
```

## Script

“借势不是风口焦虑，而是判断时代、结构、入口是否正在放大你的原力。”

## Evidence Boundary

当前为 teaching-ready。需要 3 个真实案例完成势能地图并进入 B2/B3 才算 validated teaching。
