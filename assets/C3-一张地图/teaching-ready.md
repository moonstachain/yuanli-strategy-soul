# C3 · 一张地图 · Teaching-ready

PMO Issue: #133 / #134
Status: teaching-ready + CBM-PMO backbone

## Learner Outcome

学员能把文脉、大脑、市场验证和创业结果，组织成可判断方向、优先级、能力缺口和行动路径的战略地图 / 能力组件地图。

## 30-60 min Teaching Flow

1. 一张地图不是思维导图。
2. 从模块地图到能力组件地图。
3. 识别战略控制点、支撑能力、瓶颈能力。
4. Gap Score：重要性、紧迫性、证据、可逆性、回写价值、商业影响。
5. 从 C3 Gap 生成 C4 Task。

## Correct Case

CBM-PMO 把 `strategy.direct.os-governance-core` 等 Cell 映射成 C3 Gap、C4 Task、Evidence 和 Backwrite。

## Anti-pattern

做漂亮信息图、概念地图或模块罗列，但无法回答“先做什么、什么不做、瓶颈在哪”。

## Assignment

完成：

```text
能力组件地图 + Top 3 Gap + next P0
```

## Script

“地图的价值不是告诉你这里有什么，而是告诉你什么更重要、先做什么、什么不能做。”

## Evidence Boundary

C3 已有 CBM-PMO backbone。需要更多真实客户项目输入驱动 Gap 生成，才能进入 validated operating。
