# A3 · 获得原力 · Teaching Outline

PMO Issue: #88
Asset Type: teaching-outline

## 一句话目标

帮助学员把母体中的长期能量，转化为可行动、可表达、可验证的能力系统。

## 适用对象

- 已经识别母体并完成初步回归的人
- 有能力但无法稳定输出的人
- 有经验但没有方法化的人
- 想把个人能力转成可复用资产的人

## 学完以后，学员应能完成

1. 从母体中提取核心能力。
2. 区分天赋感、技能和原力能力。
3. 把经验转成方法。
4. 设计一个小规模验证动作。
5. 写出 A3 原力能力卡。

## 课程结构

### 1. 原力不是情绪能量

原力不是兴奋、冲动或短期热情，而是长期稳定、可反复调用的能力源。

### 2. 从母体到能力

```text
母体线索 -> 核心能力 -> 方法化表达 -> 小规模验证 -> 资产沉淀
```

### 3. 三类能力

- 自然能力：不用强迫也能做
- 结构能力：能解释问题和设计路径
- 交付能力：能给他人带来结果

### 4. 经验方法化

第一性问题：我过去反复有效的做法，能否被写成步骤？

### 5. 小规模验证

第一性问题：怎样用最小动作证明这个能力对别人有价值？

## 输出作业

```text
A3 原力能力卡
```

字段：

- 我的核心能力
- 它来自哪个母体线索
- 它解决什么问题
- 它如何被方法化
- 它如何被验证
- 它可以沉淀成什么资产

## 验收标准

- 能从母体推导能力
- 能把经验写成方法
- 有最小验证动作
- 能进入 A4“显化原力”
