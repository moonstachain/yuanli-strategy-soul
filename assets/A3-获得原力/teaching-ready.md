# A3 · 获得原力 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能把母体外化成可调用的判断力、表达力、创造力、连接力和行动力，并形成第一个原力能力包。

## 30-60 min Teaching Flow

1. 母体不是能力，必须经过训练才能调用。
2. 判断外化：把脑内判断写成标准。
3. 语言外化：把经验写成表达模板。
4. 作品外化：形成样例。
5. 流程外化：让 AI / 团队可复用。
6. 反馈训练：用真实任务修正能力包。

## Correct Case

把“复杂商业判断”外化成战略诊断提纲、用户访谈问题、评分卡和样例库。

## Anti-pattern

学更多工具、写几篇内容，就以为已经获得原力。

## Assignment

完成：

```text
原力能力包 + 母体说明书 + AI 外化样例
```

## Script

“获得原力不是开悟，而是让你的源头能在真实任务里被调用。”

## Evidence Boundary

当前为 teaching-ready。需要至少 3 个真实任务调用能力包并产生反馈，才能进入 validated teaching。
