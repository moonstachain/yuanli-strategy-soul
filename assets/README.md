把 12 个正典模块从概念结构转化为可生产、可教学、可验证、可回写的内容资产。

## Standard Module Files

每个模块最终应包含：

```text
module-canon.md        正典定义
teaching-outline.md    课程大纲
cases.md               案例库
assignments.md         作业与练习
scripts.md             讲稿 / 短视频脚本
evidence.md            真实验证
evolution.md           迭代记录
```
