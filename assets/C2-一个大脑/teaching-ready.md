# C2 · 一个大脑 · Teaching-ready

PMO Issue: #133 / #134
Status: teaching-ready + real-task validation in progress

## Learner Outcome

学员能把知识、案例、术语、样例、数据和任务结果，沉淀成可检索、可调用、可版本化、可回写的 Markdown Brain。

## 30-60 min Teaching Flow

1. 一个大脑不是文件夹。
2. 什么值得记住：知识、案例、术语、样例、数据、任务结果。
3. 索引层：如何让下一次不从零开始。
4. 回写层：case -> evidence -> evolution -> canonical candidate。
5. 查询验证：检索证据、形成判断、识别缺口、生成 C4 行动。

## Correct Case

B2 外部验证从 llm-wiki 逐字稿进入 user-language、evidence、scorecard 和 writeback log。

## Anti-pattern

资料收藏很多，但没有索引、没有证据、没有回写、没有下一次复用。

## Assignment

完成：

```text
C2 查询记录 + 回写记录 + evidence/evolution 归档
```
