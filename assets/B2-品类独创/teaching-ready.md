# B2 · 品类独创 · Teaching-ready

PMO Issue: #133 / #132
Status: teaching-ready + external validation started

## Learner Outcome

学员能把原力翻译成目标用户能理解、能记住、愿意付费、愿意转述并能区别旧方案的新市场入口。

## 30-60 min Teaching Flow

1. 为什么不要只做更好的同类。
2. 旧品类缺口：用户为什么不满足？
3. 新用户 × 新问题 × 新心智位置。
4. 命名测试：简单、相关、不同、可复述。
5. 用户语言：先记录原话，再做模型解释。
6. 最小验证：5 个用户、3 句复述、1 个付费/深入了解信号。

## Correct Case

“安全价值证明”“CEO 战略仪表盘”“精准建模路径”等从真实用户语言中浮现，而不是创始人自嗨命名。

## Anti-pattern

把品类独创理解成起一个漂亮名字、包装卖点或比竞品更好一点。
