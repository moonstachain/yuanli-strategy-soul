# B2 · 品类独创 · Live Validation Checklist

PMO Issue: #130
Asset Type: live-validation-checklist
Date: 2026-06-30

## Purpose

把 B2 小范围验证工作坊推进到可执行状态。这个 checklist 用于开场前、现场中、结束后的检查。

## Before Workshop

### Participants

```text
[ ] 已筛选 5-10 个目标用户
[ ] 每人有基本角色记录
[ ] 每人有当前问题记录
[ ] 每人愿意提供反馈和复述
```

### Materials

```text
[ ] workshop-runbook.md
[ ] participant-screening.md
[ ] workshop-scorecard.md
[ ] naming-test.md
[ ] user-language.md
[ ] validation-records.md
[ ] 3 个候选名称
[ ] B2 品类独创卡模板
```

### Recording

```text
[ ] 记录用户原话
[ ] 记录候选名称反应
[ ] 记录可复述句
[ ] 记录混淆点
[ ] 记录继续了解信号
```

## During Workshop

### Must Capture

```text
[ ] 至少 5 个命名样本
[ ] 至少 5 条用户原话
[ ] 至少 3 个可复述的一句话定义
[ ] 至少 1 个继续了解或付费信号
```

### Facilitation Rule

```text
先听用户原话，不急着解释模型。
先测复述，不急着说服。
先记录混淆，不急着优化命名。
```

## After Workshop

### Writeback

```text

[ ] 更新 assets/B2-品类独创/evidence.md

```

### Decision

```text
[ ] 判断 B2 Validation 是否 1/5 -> 2/5
[ ] 判断是否需要命名迭代
[ ] 判断是否需要 offer 迭代
[ ] 判断下一个 P0 模块
```

## Pass / Fail

Pass:

```text
用户能理解、能复述、有继续了解信号。
```

Fail:

```text
用户听不懂、复述不了、只会归入旧品类。
```
