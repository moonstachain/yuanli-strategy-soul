# B2 · 品类独创 · Naming Test

PMO Issue: #129
Asset Type: naming-test
Date: 2026-06-30

## Purpose

验证 B2 品类独创是否能从“内部命名”进入“用户可理解、可复述、可传播”的新品类表达。

## Core Question

```text
这个名字是否能让目标用户立刻理解：这是为我这种人解决这个新问题的？
```

## Test Principles

1. 不测试创始人是否喜欢。
2. 测试用户是否能理解。
3. 不只测试好听。
4. 测试是否能复述。
5. 测试是否能区分旧品类。

## Naming Candidate Template

```md
# Naming Candidate

## Name

## One-line Definition

## Target User

## New Problem

## Old Category It Breaks From

## New Mental Position

## Why It Matters Now
```

## User Test Questions

对每个候选名称，问 5 个问题：

```text
1. 你觉得这个名字在解决什么问题？
2. 你觉得它是给谁的？
3. 你会把它归到哪个旧品类？
4. 你能不能用自己的话复述一句？
5. 你觉得它和常见同类有什么不一样？
```

## Scoring

| Dimension | Score 1-5 | Meaning |
|---|---:|---|
| Clarity |  | 是否一听就懂 |
| Relevance |  | 是否和目标用户相关 |
| Difference |  | 是否区别于旧品类 |
| Repeatability |  | 是否能被复述 |
| Desire |  | 是否引发进一步了解 |

## Pass Criteria

一个名称进入下一轮，需要达到：

```text
Clarity >= 4
Relevance >= 4
Difference >= 3
Repeatability >= 4
Desire >= 3
```

## Minimum Trial

```text
5-10 个目标用户
3 个候选名称
每个用户至少复述 1 次
记录原话
```

## Output

```text
winning name
user language notes
failed names
confusion points
next iteration
```
