# B2 · 品类独创 · Participant Screening

PMO Issue: #130
Asset Type: participant-screening
Date: 2026-06-30

## Purpose

筛选适合参加 B2 品类独创小范围验证工作坊的目标用户，确保反馈来自真正有问题、有判断力、有付费可能的人。

## Ideal Participant

```text
专家型个体
早期创业者
传统行业转型者
高认知内容创业者
有明确付费能力
正在思考定位、品类、产品化或增长问题
```

## Screening Questions

### 1. Current Role

```text
你现在的主要身份是什么？
你目前服务什么用户？
你当前主要收入来源是什么？
```

### 2. Current Problem

```text
你现在最想解决的定位/产品/增长问题是什么？
这个问题持续多久了？
如果不解决，会有什么损失？
```

### 3. Category Awareness

```text
你现在会把自己/产品/服务归到哪个品类？
这个旧品类有什么问题？
你是否觉得自己需要一个新的表达？
```

### 4. Buying Power

```text
你过去一年为类似问题付过费吗？
付费金额大概是多少？
如果一个工作坊能帮你讲清新品类，你愿意投入多少时间和预算？
```

### 5. Language Quality

```text
你能否用 1 分钟讲清自己现在的问题？
你是否愿意在工作坊里公开反馈和复述？
```

## Scoring

| Dimension | Score 1-5 | Meaning |
|---|---:|---|
| Problem Urgency |  | 问题是否紧迫 |
| Category Pain |  | 是否被旧品类困住 |
| Expressive Ability |  | 能否表达清楚原话 |
| Buying Power |  | 是否有付费能力 |
| Feedback Quality |  | 是否能提供高质量反馈 |

## Selection Rule

优先邀请：

```text
Problem Urgency >= 4
Category Pain >= 3
Expressive Ability >= 3
Buying Power >= 3
Feedback Quality >= 4
```

## Exclusion Rule

不优先邀请：

```text
没有明确问题
只想听课不愿反馈
没有表达能力
没有付费或行动意愿
与 B2 品类独创问题无关
```

## Minimum Group Mix

建议 5-10 人中包含：

```text
2-3 个专家型个体
2-3 个早期创业者
1-2 个传统行业转型者
1-2 个内容/IP 创业者
```
