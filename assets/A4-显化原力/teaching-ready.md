# A4 · 显化原力 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能把原力放进真实世界，判断是否有人靠近、相信、付费、复用、改变行动并愿意共同建设。

## 30-60 min Teaching Flow

1. 显化不是规模化商业化。
2. 六类验证：靠近、相信、付费、复用、行动、共建。
3. 把作品 / 样例 / 小工具放到真实用户面前。
4. 记录市场验证，而不是只记录自我感受。
5. 生成创业输入包。

## Correct Case

一个专家把原力能力包做成 5 人试跑工作坊，拿到原话、付费意向和复用请求。

## Anti-pattern

把曝光量、点赞、自己满意，误认为市场验证。

## Assignment

完成：

```text
市场验证记录 + 天选用户线索 + 贵问题线索 + 创业输入包
```

## Script

“显化原力不是让更多人看到你，而是验证世界是否真的需要这个原力。”

## Evidence Boundary

当前为 teaching-ready。需要至少一个真实用户验证闭环后进入 validated teaching。
