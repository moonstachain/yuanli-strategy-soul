# B3 · 模式升维 · Teaching Outline

PMO Issue: #92
Asset Type: teaching-outline

## 一句话目标

帮助学员把单点产品、服务或内容升级为更高杠杆的商业模式、交付系统和增长结构。

## 适用对象

- 已有新品类定位的人
- 有单点产品但交付很重的人
- 有收入但增长不可复制的人
- 想从项目型收入升级为系统型收入的人

## 学完以后，学员应能完成

1. 识别当前模式的低杠杆环节。
2. 设计前链路、后链路和财链路。
3. 区分产品、服务、系统和平台。
4. 找到模式升维的第一步。
5. 写出 B3 模式升维卡。

## 课程结构

### 1. 为什么品类成立后还需要模式升维

新品类解决心智位置，模式升维解决增长、交付和利润结构。

### 2. 三条链路

```text
前链路：获客与信任
后链路：交付与复购
财链路：收入、成本、利润和现金流
```

### 3. 杠杆诊断

第一性问题：当前增长最依赖谁的个人时间？

### 4. 模式升维路径

```text
单点服务 -> 标准产品 -> 系统交付 -> 生态协作
```

### 5. 最小升维动作

选择一个环节，让它从个人经验变成可复用系统。

## 输出作业

```text
B3 模式升维卡
```

字段：

- 当前模式
- 前链路问题
- 后链路问题
- 财链路问题
- 最大低杠杆环节
- 第一升维动作
- 预期指标

## 验收标准

- 能画出当前模式
- 能识别低杠杆环节
- 有第一升维动作
- 有指标
- 能进入 B4“壁垒锁定”
