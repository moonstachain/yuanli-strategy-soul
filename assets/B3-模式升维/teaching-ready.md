# B3 · 模式升维 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能把新品类中的价值承诺，装进前链路、后链路、财链路、产品阶梯和人机分工，形成持续赚钱、可复制、可放大的利润容器。

## 30-60 min Teaching Flow

1. 模式升维不是卖更多时间。
2. 前链路：内容、信任、获客、转化。
3. 后链路：交付、复购、客户成功、案例沉淀。
4. 财链路：收入、毛利、现金流、杠杆。
5. 产品阶梯：免费、小产品、核心产品、高阶交付。
6. 人机分工：人做判断，AI 做放大。

## Correct Case

高客单服务通过 AI 工作台，把方案呈现、客户管理、交付流程和复购场景装进同一个利润容器。

## Anti-pattern

只涨价、堆产品、追流量，或继续卖创始人的人肉时间。

## Assignment

完成：

```text
三链路画布 + 产品阶梯 + 利润容器草图
```

## Script

“模式升维不是更努力赚钱，而是把价值承诺装进一个能持续赚钱的容器。”

## Evidence Boundary

当前为 teaching-ready。需要至少 2 个真实业务案例跑出三链路和毛利/复购假设，才进入 validated teaching。
