# C1 · 一纸文脉 · Teaching Outline

PMO Issue: #94
Asset Type: teaching-outline

## 一句话目标

帮助学员把“我是谁、为什么做、边界、风格和长期承诺”写成 AI、团队和未来自己都能调用的最小正典。

## 适用对象

- 已完成原力资产与原力创业初步闭环的人
- 希望让 AI 更懂自己的人
- 希望团队协作不失真的创始人
- 想把个人战略沉淀为长期系统的人

## 学完以后，学员应能完成

1. 写出自己的身份主线。
2. 明确长期承诺与不做边界。
3. 提炼表达风格和判断原则。
4. 写出 AI 可调用的一纸文脉。
5. 建立后续 OS 的正典入口。

## 课程结构

### 1. 为什么需要一纸文脉

没有文脉，AI 会变成工具；有文脉，AI 才可能成为共同进化的系统接口。

### 2. 一纸文脉的六个字段

```text
身份主线 / 长期承诺 / 核心判断 / 风格边界 / 不做清单 / 回写规则
```

### 3. 从个人故事到系统上下文

第一性问题：什么信息必须被 AI 和团队长期记住？

### 4. 文脉不是简介

简介给别人看，文脉给系统调用。

### 5. 文脉回写

每一次重要行动之后，都要判断是否需要更新文脉。

## 输出作业

```text
C1 一纸文脉卡
```

字段：

- 我是谁
- 我为什么做
- 我的长期承诺
- 我的判断原则
- 我的风格边界
- 我的不做清单
- 我的回写规则

## 验收标准

- 不是个人简介
- 能指导 AI 判断
- 能帮助团队减少误解
- 有边界
- 能进入 C2“一个大脑”
