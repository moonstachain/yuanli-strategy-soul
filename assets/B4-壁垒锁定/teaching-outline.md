# B4 · 壁垒锁定 · Teaching Outline

PMO Issue: #93
Asset Type: teaching-outline

## 一句话目标

帮助学员把增长成果沉淀为品牌、关系、数据、流程、组织和生态壁垒，避免增长之后重新归零。

## 适用对象

- 已经有产品、用户和收入的人
- 增长后容易被复制的人
- 交付依赖个人的人
- 想把业务沉淀成长期资产的人

## 学完以后，学员应能完成

1. 识别当前业务已有壁垒和缺失壁垒。
2. 区分短期优势和长期壁垒。
3. 设计至少一个可沉淀壁垒。
4. 建立壁垒指标。
5. 写出 B4 壁垒锁定卡。

## 课程结构

### 1. 为什么增长不等于壁垒

增长可以来自机会、流量或执行，但壁垒来自可持续的控制点和复利资产。

### 2. 六类壁垒

```text
品牌 / 关系 / 数据 / 流程 / 组织 / 生态
```

### 3. 短期优势 vs 长期壁垒

第一性问题：如果竞争者复制我的动作，我还剩什么？

### 4. 控制点设计

第一性问题：我的业务中哪个关键环节必须由我掌握？

### 5. 壁垒指标

```text
复购率 / 转介绍 / 数据积累 / SOP 完成度 / 团队复制率 / 生态伙伴数量
```

## 输出作业

```text
B4 壁垒锁定卡
```

字段：

- 当前优势
- 可被复制的部分
- 不可轻易复制的部分
- 关键控制点
- 壁垒类型
- 壁垒动作
- 壁垒指标

## 验收标准

- 不把短期增长误认为壁垒
- 有明确控制点
- 有可沉淀资产
- 有壁垒指标
- 能进入 C1“一纸文脉”
