# B4 · 壁垒锁定 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能把已跑通的利润容器，拆成虚壁垒、实壁垒、入壁垒、出壁垒、飞轮壁垒和母体壁垒，并形成 16 控制点。

## 30-60 min Teaching Flow

1. 壁垒不是口号，也不是保密。
2. 虚壁垒：认知、品牌、意义、叙事、品味。
3. 实壁垒：产品、交付、数据、工具、供应链。
4. 入壁垒：入口、渠道、社群、信任。
5. 出壁垒：复购、迁移成本、身份绑定。
6. 飞轮壁垒：内容、案例、数据互相强化。
7. 母体壁垒：复制形式仍缺源头。

## Correct Case

把客户案例、行业数据、工具流程、内容表达和用户身份绑定，沉淀为持续强化的财富飞轮。

## Anti-pattern

先发幻觉、自我安慰式护城河、只靠品牌口号或保密。

## Assignment

完成：

```text
壁垒地图 + 16 控制点 + 财富飞轮
```

## Script

“真正的壁垒不是别人看不见你在做什么，而是看见了也拿不走。”
