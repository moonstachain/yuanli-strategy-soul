# A1 · 发现母体 · Teaching-ready

PMO Issue: #133
Status: teaching-ready skeleton

## Learner Outcome

学员能从高能事件、低能事件、反复判断、客户找他的理由中，提出一个可验证的“原力母体候选”。

## 30-60 min Teaching Flow

1. AI 清算：哪些能力会被平权？
2. 重复差异：哪些判断、作品、关系、意义持续从你身上长出来？
3. 事件萃取：高能 / 低能 / 至暗 / 被反复求助。
4. 母题浮现：把重复线索压缩成 3 个母体候选。
5. 一句话假设：我持续生成差异的源头是……

## Correct Case

客户反复找一个创业者，不是因为他“懂营销”，而是因为他总能把复杂局面压缩成可执行战略路径。母体候选不是营销，而是“复杂系统翻译成行动秩序”。

## Anti-pattern

把兴趣、MBTI、职业标签、别人夸奖的能力清单直接当成母体。

## Assignment

完成：

```text
母体候选清单 + 同质化能力清算表 + 下一步返源问题
```

## Script

“我们不是找你会什么，而是找 AI 清算之后，你身上仍然持续生成差异的那个源头。”

## Evidence Boundary

当前为 teaching-ready。需要真实学员完成 5 份母体候选卡后，才能进入 validated teaching。
