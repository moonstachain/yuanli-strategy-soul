## 一句话定义

一纸文脉，是把我是谁、为什么做、原力母体、长期承诺、品味函数、战略边界和表达风格，写成 AI、团队与未来自己都能理解和调用的最小正典文件。

## 不是它

```text
不是个人简介。
不是品牌文案。
不是愿景口号。
不是提示词集合。
不是堆满资料的长文档。
不是让 AI 平均化模仿我的话术。
```

## 第一性问题

```text
AI 如何真正懂我，并持续保持我的判断边界？
```

## 原子模块

```text
C1.1 母体说明：我的原力母体是什么。
C1.2 长期承诺：我愿意用十年服务什么。
C1.3 品味函数：什么值得做，什么不值得做。
C1.4 战略边界：什么不做、服务谁、不服务谁。
C1.5 语言风格：常用概念、隐喻、表达方式、审美偏好。
C1.6 禁止事项：AI 与团队不得越界的内容。
C1.7 AI 使用协议：AI 如何调用文脉、何时必须请求人类判断。
```

## 输入

```text
A2 原力母体定义
A3 我的说明书 V3
A4 创业验证记录
B4 OS 回写包
代表性表达样本
战略边界
用户语言与反馈
```

## 判断标准

```text
AI 是否能基于它稳定判断？
是否知道什么不该做？
是否能保持口径一致？
是否足够短，能被持续维护？
是否能为 C2 大脑、C3 地图、C4 链路提供约束？
是否能让任务不偏离母体？
```

## 输出

```text

我的说明书 V3
AI 使用协议
战略边界清单
风格词典
禁止事项清单

```
