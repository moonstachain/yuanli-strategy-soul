## 一句话定义

一张地图，是把文脉、大脑、市场验证和创业结果，组织成一张可判断方向、取舍、优先级、能力缺口和行动路径的战略地图 / 能力组件地图 / 事业系统地图。

## 不是它

```text
不是思维导图。
不是战略口号。
不是信息可视化。
不是漂亮但不能决策的图。
不是把概念画成一堆节点。
不是没有行动阈值的展示页。
```

## 第一性问题

```text
系统如何知道什么更重要、先做什么、如何取舍、能力缺口在哪里？
```

## 原子模块

```text
C3.1 目标地图：当前最重要目标是什么。
C3.2 用户地图：服务谁、不服务谁、心理账户是什么。
C3.3 机会地图：母体与势能如何匹配。
C3.4 因果地图：关键变量如何互相影响。
C3.5 商业地图：链路、模式、壁垒在哪里。
C3.6 能力组件地图：战略控制点、支撑能力、瓶颈能力、可外包能力、必须自持能力。
C3.7 决策阈值：什么条件下推进、暂停、放弃。
```

## 输入

```text
C1 一纸文脉
C2 一个大脑
A4 市场验证记录
B1-B4 创业四关结果
任务复盘
战略目标
客户成功案例
数据指标
```

## 判断标准

```text
是否能帮助取舍？
是否能解释因果？
是否能指导下一步行动？
是否能识别战略控制点？
是否能识别能力缺口？
是否能被验证和更新？
是否能生成 C4 的行动链路？
```

## 输出

```text
一张战略地图
能力组件地图
事业系统地图
机会地图
商业地图
变量因果图
行动阈值
战略路径图
C4 一条链路输入
```
