## 一句话定义

一个大脑，是把知识、案例、术语、样例、数据、经验和任务结果，沉淀成可检索、可调用、可版本化、可更新的记忆系统。

## 不是它

```text
不是文件夹堆积。
不是资料收藏。
不是知识库幻觉。
不是没有版本的杂物箱。
不是每次任务都从零开始的资料堆。
不是没有文脉约束的通用数据库。
```

## 第一性问题

```text
系统应该记住什么，才能让下一次不从零开始？
```

## 原子模块

```text
C2.1 知识库：理论、资料、研究、书稿。
C2.2 案例库：客户、项目、交付、成功与失败。
C2.3 术语库：原力体系概念定义、别名、边界。
C2.4 样例库：好输出、坏输出、像我、不像我。
C2.5 数据库：指标、反馈、运营数据、验证记录。
C2.6 索引层：让 AI 与人知道从哪里找。
C2.7 版本层：每个知识资产有来源、状态、更新时间和可信度。
```

## 输入

```text
C1 一纸文脉
书稿
源稿
项目案例
客户反馈
模板填写稿
市场数据
任务输出
回写记录
```

## 判断标准

```text
是否可检索？
是否可信？
是否有版本？
是否能被下一次任务调用？
是否和文脉一致？
是否能区分正典、草稿、历史、参考？
是否支持 C3 地图生成？
```

## 输出

```text
一个大脑
知识索引
案例索引
术语表
样例库
数据记录
版本记录
C3 一张地图输入
```
