## 一句话定义

品类独创，是把被时代势能放大的原力，翻译成目标用户能理解、能记住、愿意付费、愿意转述，并能与旧方案区分的新市场入口。

## 不是它

```text
不是起一个好听名字。
不是包装卖点。
不是比竞品更好。
不是只做差异化话术。
不是在旧品类里卷参数。
不是只追求“别人觉得高级”。
```

## 第一性问题

```text
市场如何认得我？
```

## 原子模块

```text
B2.1 甜用户：谁最自然需要这个原力。
B2.2 贵任务：什么问题值得用户付费解决。
B2.3 心理账户：用户把这件事放在哪个预算和意义账户。
B2.4 旧品类对标：用户原来如何理解这类问题。
B2.5 新品类命名：我不是旧方案，而是什么新方案。
B2.6 入脑表达：一句话让人理解、记住、转述。
B2.7 用户语言库：沉淀用户原话，而不是自嗨表达。
```

## 输入

```text
B1 势能地图
机会假设
天选用户线索
贵问题线索
心理账户线索
用户原话
旧方案 / 竞品
真实互动记录
```

## 判断标准

```text
用户是否一听就懂？
用户是否能转述？
用户是否知道为什么付费？
是否能把我和旧方案区分开？
是否进入更高价值的心理账户？
是否能生成 B3 的价值承诺与产品阶梯输入？
```

## 输出

```text
新品类定义
一句话价值主张
心理账户定位
品类命名候选
用户语言库
内容表达框架
B3 模式升维输入
```
