## 一句话定义

模式升维，是把被市场认出的原力品类，升级为可持续赚钱、可复制、可扩张，并能沉淀数据与交付资产的利润容器。

## 不是它

```text
不是卖更多时间。
不是简单涨价。
不是堆产品。
不是只追流量。
不是只有收入没有毛利。
不是增长越快人肉越崩溃。
```

## 第一性问题

```text
如何持续赚钱、可复制？
```

## 原子模块

```text
B3.1 前链路：获客、信任、内容、转化。
B3.2 后链路：交付、复购、客户成功、案例沉淀。
B3.3 财链路：收入结构、毛利结构、现金流、杠杆。
B3.4 产品阶梯：免费内容、小产品、核心产品、高阶交付。
B3.5 数据反馈：每轮交付如何产生下一轮增长资产。
B3.6 人机分工：哪些由人做，哪些由 AI / agent / SOP 承担。
B3.7 利润容器：把价值承诺装成能持续蓄水的商业结构。
```

## 输入

```text
B2 新品类定义
目标用户
一句话价值主张
心理账户定位
内容表达框架
交付能力
价格假设
渠道与内容资产
```

## 判断标准

```text
增长是否导致人肉崩溃？
是否有可复用交付？
是否有清晰毛利和现金流？
是否有复购、转介绍与数据沉淀？
是否有人机分工与 SOP 化空间？
是否能进入 B4 壁垒锁定？
```

## 输出

```text
利润容器
产品阶梯
三链路画布
交付 SOP
增长指标
人机分工表
B4 壁垒锁定输入
```
