## 一句话定义

原力借势，是判断这个时代为什么现在需要我的原力，并找到最适合进入市场的势能窗口。

## 不是它

```text
不是追风口。
不是蹭热点。
不是宏观叙事自嗨。
不是把母体强行塞进不适合的趋势。
不是重新展开“找你是谁”。
不是把 A 轴的合力内容搬回创业关一。
```

## 第一性问题

```text
这个时代为什么现在需要我？
```

## 原子模块

```text
B1.1 宏周期势：康波、技术范式、资本周期、AI 渗透率。
B1.2 中结构势：政策、经济、社会、技术、产业价值链。
B1.3 微入口势：用户心智、心理账户、渠道窗口、预算迁移。
B1.4 合力匹配：我的原力是否刚好适配这股势。
B1.5 时间窗口：早、刚好、晚，分别对应不同打法。
B1.6 势能排序：多个机会如何排序。
B1.7 全局最优下注：不选热闹机会，选最适合母体的结构性机会。
```

## 输入

```text
A4 创业输入包
市场验证记录
天选用户线索
贵问题线索
心理账户线索
行业趋势资料
用户变化信号
政策 / 技术 / 渠道变化
```

## 判断标准

```text
是否只是热闹风口？
是否存在中长期结构变化？
是否与我的原力同向？
是否有明确入口和预算迁移？
是否存在可进入的时间窗口？
是否能生成 B2 的品类独创输入？
```

## 输出

```text
势能地图
机会假设
入场窗口判断
机会优先级
原力借势卡
B2 品类独创输入
```
