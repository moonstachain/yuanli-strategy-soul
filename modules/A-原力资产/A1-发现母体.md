## 一句话定义

发现母体，是从同质化能力、外部标签、职业惯性和二手共识中，识别出自己持续生成判断、作品、关系和意义的底层生命结构。

## 不是它

```text
不是兴趣爱好。
不是人格标签。
不是职业优势。
不是人设包装。
不是短期灵感。
不是“我会什么”的技能清单。
```

## 第一性问题

```text
AI 清算所有同质化能力之后，我身上还剩什么会持续生成差异？
```

## 原子模块

```text
A1.1 同质化能力清算：哪些能力正在被 AI 平权、复制、自动化。
A1.2 算力活 / 原力活分堆：过去一周哪些任务只是信息处理，哪些任务来自不可复制的我。
A1.3 异常注意：我天然会注意什么，别人看不见什么。
A1.4 高能事件：什么事情让我越做越清楚、越做越有能量。
A1.5 低级答案厌恶：我最无法忍受哪些粗糙、伪高级或错误答案。
A1.6 他人获得物：别人反复从我这里获得什么。
A1.7 母题浮现：这些现象背后共同指向的主问题是什么。
```

## 输入

```text
过去三年的高能事件
过去三年的低能事件
过去一周工作清单
别人反复找我解决的问题
我长期关注的主题
我最不能忍受的低级答案
AI 已经可以替代的工作清单
代表性表达样本
```

## 判断标准

```text
它是否长期稳定？
它是否不依赖单一技能？
它是否解释了我过去多个选择？
它是否持续生成判断、作品、关系或意义？
它是否是 AI 难以直接复制的生命结构？
它是否能产生下一步的回到母体动作？
```

## 输出

```text
母体候选清单
母题清单
同质化能力清单
算力活 / 原力活清单
原力活占比
母体一句话假设
下一步返源问题
```
