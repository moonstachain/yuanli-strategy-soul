## 一句话定义

显化原力，是把已经获得的原力放进真实世界，验证它是否被人需要、相信、复用、付费和共同建设，并生成进入《原力创业》的输入包。

## 不是它

```text
不是完整商业模式。
不是规模化变现。
不是创业全部。
不是把原力廉价卖掉。
不是用户口头喜欢。
不是没有行为证据的自我确认。
```

## 第一性问题

```text
这个原力是否真的被世界需要？
```

## 原子模块

```text
A4.1 天选用户：谁最自然被这个原力吸引。
A4.2 贵问题：他们有什么不解决代价很高的问题。
A4.3 心理账户线索：他们把这个价值放在哪个预算与意义账户。
A4.4 最小显化：用内容、咨询、课程、工具、服务原型进行真实验证。
A4.5 市场六问：靠近、相信、付费、复用、行动、共建。
A4.6 真实行为证据：付款、转介绍、复用、主动反馈、共同建设。
A4.7 创业输入包：把验证结果交给 B1 原力借势。
```

## 输入

```text
A3 原力能力包
作品样本
目标人群假设
真实互动记录
初始付费 / 复用 / 转介绍证据
内容反馈与用户原话
```

## 判断标准

```text
是否有真实行为，而不是口头喜欢？
是否出现天选用户？
是否显现贵问题？
是否出现心理账户线索？
是否值得进入创业四关？
是否能回写母体？
是否能生成 B1 的势能判断输入？
```

## 输出

```text
天选用户线索
贵问题线索
心理账户线索
市场验证记录
最小显化样本
创业输入包
母体回写记录
```
