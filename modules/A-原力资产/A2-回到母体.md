# A2 · 回到母体

> 来源：`docs/07-原力战略三部曲-龙骨总纲.md`、`yuanli-strategy-trilogy/trilogy/原力资产-基石与关键概念-全谱.md`

## 一句话定义

回到母体，是停止被外部评价、短期机会、工具兴奋和路径惯性牵引，重新以原力母体为中心组织注意力、品味、能力、经验、关系和长期承诺。

## 不是它

```text
不是退回舒适区。
不是自我感动。
不是排斥市场。
不是只做喜欢的事。
不是逃避商业化。
不是把母体包装成一句漂亮文案。
```

## 第一性问题

```text
如果我的人生、事业和 AI 系统都以母体为中心，哪些事情要保留，哪些事情要停止？
```

## 原子模块

```text
A2.1 注意力雷达：我天然会看见什么。
A2.2 品味函数：我如何判断什么值得做，什么不值得做。
A2.3 身体化能力：我长期训练出的手感、嗅觉、判断和行动回路。
A2.4 生命经验：哪些经历不可逆地塑形了我。
A2.5 长期承诺：我愿意用十年持续服务什么。
A2.6 边界与弃绝：哪些机会、关系、任务不再服务母体。
A2.7 母体重组：用母体重新组织选择、项目、关系、AI 上下文。
```

## 输入

```text
A1 母体候选
注意力事件
品味样本
能力案例
关键生命经历
高能 / 低能关系
长期想服务的问题
当前项目清单
当前不再做清单草稿
```

## 判断标准

```text
五部件是否互相支撑？
注意力是否能解释品味？
能力是否来自长期反馈？
生命经验是否指向长期承诺？
边界是否足够清楚？
是否能明确放弃不服务母体的机会？
是否能进入 A3 的能力外化？
```

## 输出

```text
原力母体定义
注意力雷达卡
品味函数卡
身体化能力卡
生命经验炼金卡
长期承诺卡
不再做清单
母体重组原则
```
