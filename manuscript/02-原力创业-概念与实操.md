## 02.1 创业不是从零造势

很多人理解创业，是从零开始：从零想一个点子，从零做一个产品，从零找客户，从零造一个市场。

这听起来很英雄主义。

但大多数人死在这里。

因为真正的财富，不是靠你一个人硬造出来的。财富来自乘法：你有一个能繁殖的母体，又把它放进一个会自己变大的环境里。

这就是《原力创业》的第一性公式：

```text
财富 = 借势 × 合力
```

借势，是外部不变量。

合力，是内部不变量。

借势解决：时代往哪走？技术往哪走？产业往哪走？用户心智往哪走？

合力解决：我是谁？我的原型力是什么？我的非对称优势、非理性痴迷、非线性回报在哪里？

两者是乘法，不是加法。

有合力但没势，会怀才不遇；有势但没合力，会成为风口炮灰。任一为零，整体归零。

所以，原力创业不是让你从零造势，而是让你识别已有的势，把你的原力资产放进去。

它要完成的，是一句话：

> **把做自己，变成好生意。**

## 02.2 从原力资产到原力创业

第一部《原力资产》解决的是“找到你”。

你已经有了母体定义卡，知道自己的三圈交集、天才区、原型、阴影和元AI上下文。

但找到你，还不是创业。

创业必须回答另一个问题：

```text
市场为什么要为这个你付费？
```

你喜欢，不代表用户需要。

你擅长，不代表用户愿意付费。

你有意义，不代表它有商业闭环。

所以，《原力创业》要做的，是把原力资产送进四个系统：

```text
时代系统：有没有势？
用户系统：谁最甜？
现金流系统：怎么赚钱？
壁垒系统：怎么守住？
```

这四个系统，对应四个关口：

```text
原力觉醒 → 品类独创 → 模式升维 → 壁垒锁定
```

每一关过掉，都会带走一份资产：

```text
原力资产 → 心理账户 → 利润容器 → 财富飞轮
```

这就是原力创业的资产生命线。

## 02.3 四关：商业闭环的生死位

四关不是四个章节，而是创业的四个生死端口。

| 关 | 方向 | 第一性问题 | 获得物 |
|---|---|---|---|
| 原力觉醒 | 向内求 | 我能不能持续做？ | 原力资产 |
| 品类独创 | 向外立 | 市场认不认？ | 心理账户 |
| 模式升维 | 向上升 | 赚不赚得到？ | 利润容器 |
| 壁垒锁定 | 向守远 | 守不守得住？ | 财富飞轮 |

第一关，如果你不知道自己是谁，你就在卖身。你做的东西可能赚钱，但会耗尽你。

第二关，如果市场听不懂你，你就在卷价格。你再好，也会被放进旧品类里比较。

第三关，如果模式跑不通，你就在用人肉换钱。有客户、有口碑、有忙碌，但没有利润容器。

第四关，如果壁垒锁不住，你赚到的钱会被抄走，最后为他人作嫁衣。

所以原力创业的纪律很简单：

```text
不要跳关。
```

觉醒没完成，不要做品类独创。

品类没跑通，不要做模式升维。

模式没闭环，不要做壁垒锁定。

创业不是快，而是顺序正确。

## 02.4 第一关：原力觉醒

原力觉醒不是重新做一遍《原力资产》。

它是把《原力资产》的输出，变成创业第一关的通关条件。

这一关的核心公式是：

```text
原力 = 非对称优势 ∩ 非理性痴迷 ∩ 非线性回报
```

非对称优势，是别人学不来、你自然会的东西。

非理性痴迷，是不给钱你也忍不住要做的东西。

非线性回报，是一旦被 AI、资本、网络、组织或内容放大，可以扩 100 倍而不累的东西。

这三者必须同时成立。

如果只有优势，没有痴迷，那只是技能。

如果只有痴迷，没有优势，那只是兴趣。

如果优势和痴迷都有，但没有非线性回报，那只是爱好。

创业要的是原力，不是努力。

### 觉醒通关检查

```text
1. 能 30 秒说清自己的三圈交集。
2. 完成 12 原型测评，主/次原型清晰。
3. 完成天才区能量回流测试，知道天才区在哪一象限。
4. 完成 3-2-1 阴影练习，金色阴影方向清晰。
5. 输出一句话天赋定位，不超过 30 字。
```

第一关的输出不是商业计划书，而是：

```text
原力资产 / 能量原点 / 一句话天赋定位
```

## 02.5 第二关：品类独创

找到原力后，下一个问题是：市场怎么认得你？

大多数创业者会直接进入旧品类。

别人做知识付费，我也做知识付费。

别人做咨询，我也做咨询。

别人做 AI 工具，我也做 AI 工具。

一旦进入旧品类，你就会被旧标准比较：谁更便宜、谁更快、谁功能更多、谁案例更多。

这叫同质化竞争。

原力创业不从这里开始。

它从品类独创开始。

品类独创有十六字真言：

```text
无法分类 / 重新定义 / 自带预算 / 品类独创
```

你要让用户觉得：这不是我熟悉的旧东西，但它刚好重新定义了我一直说不清的问题，而且我愿意为它调动预算。

品类独创的落地公式是：

```text
三角定位 = 甜用户 × 贵任务 × 入脑表达
```

甜用户，是你最强禀赋最能服务、且愿意长期为你买单的人。

贵任务，是他们愿意高价解决的问题。这个问题最好落在情绪账户、社交账户或投资账户，而不只是功能账户。

入脑表达，是 30 秒能听懂、能复述、愿意付费的一句话。

### 品类通关检查

```text
1. 用十六字真言验证品类，四维全部通过。
2. 锁定甜用户，S级客户 ≥ 5。
3. 找到至少3个贵任务，进入情绪 / 社交 / 投资账户。
4. 30秒电梯测试通过，用户听懂且愿意复述。
5. 输出一句入脑价值主张，不超过20字。
```

第二关的输出是：

```text
心理账户 / 独创品类 / 一句入脑
```

## 02.6 第三关：模式升维

很多创业项目不是没人买，而是不能复制。

创始人很能干，客户很满意，项目很热闹，但一离开创始人，系统就停。

这不是商业模式。

这是高级个体户。

模式升维要解决的问题是：

```text
从个人能干，到系统能跑。
```

核心结构是三链贯通：

```text
前链路 × 后链路 × 财链路
```

前链路解决渴望度：用户为什么想要？怎么找到他？怎么让他愿意进入？

后链路解决可行性：你凭什么稳定交付？核心资源、关键业务、伙伴网络是否撑得住？

财链路解决生存性：收入来源、定价机制、成本结构是否成立？

这三条链路合起来，对应商业模式九模块。

```text
前链路：用户细分 / 价值主张 / 渠道结构 / 客户关系
后链路：核心资源 / 关键业务 / 伙伴网络
财链路：收入来源 / 定价机制 / 成本结构
```

原力创业还有一个判断：商业模式不是凭空发明，而是组合。

过去 300 年，人类商业验证过的基本组件可以压缩为 58 种商业模式原子。创业者要做的，不是幻想自己发明了一个全新商业，而是在自己的用户、品类、康波与能力约束下，找到最适合自己的原子组合。

### 模式通关检查

```text
1. 商业模式九模块画布填满。
2. 9张闭环验证测试卡至少3张跑通。
3. 三链路闭环跑通：获客 → 交付 → 变现一个真实订单。
4. 找到至少1个 LTV/CAC > 3 的超级单点。
5. ABCS 段位 ≥ A，能用数字说清模式。
```

第三关的输出是：

```text
利润容器 / 三链路闭环 / 可复制模式
```

## 02.7 第四关：壁垒锁定

赚到钱，不等于创业成功。

很多项目真正危险的时刻，不是没赚到钱，而是刚赚到钱。

因为一旦证明你能赚钱，模仿者、平台、巨头、同行、上下游都会靠近。没有壁垒，利润不是你的，只是暂时经过你。

壁垒锁定要回答的问题是：

```text
模式升维赚到了钱，怎么守住不被收割？
```

核心结构是：

```text
四维护城 × 16 战略控制点
虚 × 实 × 入 × 出
```

虚壁垒，是用户认你。品牌、心智、品类、文化叙事，让别人进不来。

实壁垒，是你做得到。极致成本、资产密度、链主控制、工艺黑箱，让别人追不上。

入壁垒，是用户离不开。学习成本、数据沉淀、生态迭代、沉没成本，让别人拿不走。

出壁垒，是用户互锁。用户网络、平台效应、双边市场、生态飞轮，让系统自己长。

### 壁垒通关检查

```text
1. 16控制点至少8个达到高分锚点。
2. 至少6个控制点在持续投资中。
3. 客户转换成本可量化，流失率低于目标线。
4. 至少1个壁垒达到“竞品6个月追不上”。
5. 输出一页纸“我的护城河”，含证据与30法则推演。
```

第四关的输出是：

```text
财富飞轮 / 护城河 / 穿越周期能力
```

## 02.8 四获得物：从本金到飞轮

原力创业不是每关完成一项作业，而是每关带走一份可复利资产。

```text
原力资产 → 心理账户 → 利润容器 → 财富飞轮
```

白话版是：

```text
本金 → 开户 → 容器 → 飞轮
```

第一关，原力资产是本金。你终于知道自己是谁，知道市场应该放大你的哪一部分。

第二关，心理账户是开户。你不再停留在功能价值，而是进入用户真正愿意为你调动预算的心智账户。

第三关，利润容器是容器。你不再靠人肉交付，而是把前链路、后链路、财链路接起来，让价值可以复制。

第四关，财富飞轮是飞轮。你不只是赚到钱，而是让利润被壁垒锁住，形成长期复利。

出师，就是这四个获得物同时成立。

```text
左手现金流 = 心理账户 × 利润容器的当期产出
右手资产雪球 = 财富飞轮的长期复利
出师 = 左手现金流 + 右手资产雪球
```

## 02.10 本章边界

《原力创业》只负责三件事：

```text
把原力资产放进市场。
把市场需求做成利润容器。
把利润容器锁成财富飞轮。
```

它不负责重新解码“你”的全部深层结构，那是《原力资产》。

它不负责搭 AI 班底、自动化链路、知识库和工作生命体，那是《原力OS》。

它也不负责具体投资组合和家庭资产配置，那是未来《原力财富》。

本章的终点，是一句话：

> 我知道如何把我的原力，变成一门可通关、可复利、守得住的生意。

从这里开始，我们进入第三部：

> **原力OS。**
>
> 把做事系统，变成会自我繁殖的工作生命体。
