# 02 · 原力创业：变现你

> v0.2 重编草稿  
> 目标：拿着已经确认的原力资产，找到它最该搭上的时代顺风，并把它变成可通关、可复利、守得住的生意。

---

## 02.0 本章关键裁决

本章做一次重要修正：

```text
“原力觉醒”归入《原力资产》。
“原力借势”成为《原力创业》第一关。
```

原因很简单：第一部《原力资产》已经完成了觉醒，已经生成了合力。创业这一边不应该再重复向内问“我是谁”，而应该向外问：

```text
我已经知道我是谁了。
那么，我该借哪股势，把这个“我”放大成一门生意？
```

所以新的创业四关是：

```text
原力借势 → 品类独创 → 模式升维 → 壁垒锁定
```

这不是简单改名，而是把《原力创业》的第一关从“向内重复”改成“向外开局”。

## 02.1 创业不是从零造势

很多人理解创业，是从零开始：从零想一个点子，从零做一个产品，从零找客户，从零造一个市场。

这听起来很英雄主义。

但大多数人死在这里。

因为真正的财富，不是靠你一个人硬造出来的。财富来自乘法：你有一个能繁殖的母体，又把它放进一个会自己变大的环境里。

这就是《原力创业》的第一性公式：

```text
财富 = 借势 × 合力
```

借势，是外部不变量。

合力，是内部不变量。

合力来自《原力资产》：你的三圈交集、天才区、原型力、非理性痴迷、非对称优势、非线性回报。

借势则是《原力创业》的开局：康波往哪走？技术往哪走？产业往哪走？政策、经济、社会、用户心智和预算往哪迁移？

两者是乘法，不是加法。

有合力但没势，会怀才不遇；有势但没合力，会成为风口炮灰。任一为零，整体归零。

所以，原力创业不是让你从零造势，而是让你识别已有的势，把你的原力资产放进去。

它要完成的，是一句话：

> **把做自己，变成好生意。**

## 02.2 从原力资产到原力借势

第一部《原力资产》解决的是“找到你”。

你已经有了母体定义卡，知道自己的三圈交集、天才区、原型、阴影和元AI上下文。

但找到你，还不是创业。

创业必须回答另一个问题：

```text
这个时代为什么现在需要我？
```

你喜欢，不代表用户需要。

你擅长，不代表用户愿意付费。

你有意义，不代表它刚好踩在时代窗口上。

所以，《原力创业》第一关不是重新觉醒，而是借势：

```text
拿着已经确认的原力资产，去寻找它最该搭上的时代顺风。
```

这个顺风至少有四层：

```text
宏周期势：康波、技术范式、资本周期、AI渗透率。
中结构势：政策、经济、社会、技术、产业价值链。
微入口势：用户心智、心理账户、渠道窗口、预算迁移。
合力匹配：我的原力资产是否刚好适配这股势。
```

## 02.3 四关：商业闭环的生死位

新的四关是：

```text
原力借势 → 品类独创 → 模式升维 → 壁垒锁定
```

| 关 | 第一性问题 | 获得物 |
|---|---|---|
| 原力借势 | 这个时代为什么现在需要我？ | 势能地图 |
| 品类独创 | 市场如何认得我？ | 心理账户 |
| 模式升维 | 怎么持续赚钱、可复制？ | 利润容器 |
| 壁垒锁定 | 怎么守住、不被拿走？ | 财富飞轮 |

第一关，如果你没有借势判断，你可能只是拿着合力在错误窗口里硬撞。

第二关，如果市场听不懂你，你就在卷价格。你再好，也会被放进旧品类里比较。

第三关，如果模式跑不通，你就在用人肉换钱。有客户、有口碑、有忙碌，但没有利润容器。

第四关，如果壁垒锁不住，你赚到的钱会被抄走，最后为他人作嫁衣。

所以原力创业的纪律很简单：

```text
不要跳关。
```

没有借势判断，不要做品类独创。

品类没跑通，不要做模式升维。

模式没闭环，不要做壁垒锁定。

创业不是快，而是顺序正确。

## 02.4 第一关：原力借势

原力借势要回答：

```text
我借的是哪一股势？
为什么是现在？
我的合力为什么刚好适配这股势？
```

它不是空喊“顺势而为”。

它必须有证据。

### 02.4.1 宏周期势

宏周期势看的是大时间尺度：康波、技术范式、资本周期、AI渗透率。

你要判断：

```text
我们处在旧周期尾部，还是新周期初春？
这项技术是玩具、工具，还是基础设施？
资本、人才、平台和用户注意力正在流向哪里？
```

原力创业强调“千日窗口”：每次范式转移的早期窗口，往往决定后面十年甚至二十年的品类定义权。

### 02.4.2 中结构势

中结构势看的是 PEST / PESTEL：政策、经济、社会、技术，也可以扩展到环境和法律。

你要问：

```text
P 政策：哪些政策、监管、公共议题正在改变供需？
E 经济：哪些收入、成本、资本、价格机制正在变化？
S 社会：哪些人群、代际、身份、生活方式正在迁移？
T 技术：哪些技术曲线正在进入可用、便宜、普及阶段？
```

PEST 的意义不是做咨询报告，而是帮助你看见：

```text
哪一股结构性变化，正在替我的原力资产打开一扇门？
```

### 02.4.3 微入口势

微入口势看的是入口：用户心智、心理账户、渠道窗口、预算迁移。

很多时代大势太宏大，不能直接变成生意。你必须找到一个小入口：

```text
哪类用户最先感到变化？
他们的旧方案哪里失效？
他们的预算从哪里迁移过来？
哪个渠道窗口刚刚打开？
他们现在最愿意为什么付费？
```

这一步会自然进入第二关“品类独创”。

### 02.4.4 合力匹配

最后，你要证明：这股势不是你硬蹭的。

你要把它和自己的原力资产对齐：

```text
我的非对称优势是否刚好适配这股势？
我的非理性痴迷是否能支撑长期投入？
我的非线性回报是否能被这股势放大？
我的母体定义卡是否能自然长出这个机会？
```

如果答案是否定的，再大的势也不是你的势。

### 原力借势通关检查

```text
1. 能 30 秒说清：我借的是哪一股势。
2. 能用康波 / 技术 / 产业 / PEST 至少三类证据证明这股势存在。
3. 能说明：为什么是现在，不是三年前，也不是三年后。
4. 能证明：我的原力资产和这股势同向，而不是硬蹭风口。
5. 能输出明确进入点：甜用户、贵任务、预算来源、品类机会。
```

第一关的输出不是“一句话天赋定位”，那已经在《原力资产》完成。

第一关的新输出是：

```text
势能地图
借势合力卡
千日窗口判断
机会假设卡
初始进入点
```

## 02.5 第二关：品类独创

找到势后，下一个问题是：市场怎么认得你？

大多数创业者会直接进入旧品类。

别人做知识付费，我也做知识付费。

别人做咨询，我也做咨询。

别人做 AI 工具，我也做 AI 工具。

一旦进入旧品类，你就会被旧标准比较：谁更便宜、谁更快、谁功能更多、谁案例更多。

这叫同质化竞争。

原力创业不从这里开始。

它从品类独创开始。

品类独创有十六字真言：

```text
无法分类 / 重新定义 / 自带预算 / 品类独创
```

你要让用户觉得：这不是我熟悉的旧东西，但它刚好重新定义了我一直说不清的问题，而且我愿意为它调动预算。

品类独创的落地公式是：

```text
三角定位 = 甜用户 × 贵任务 × 入脑表达
```

甜用户，是你最强禀赋最能服务、且愿意长期为你买单的人。

贵任务，是他们愿意高价解决的问题。这个问题最好落在情绪账户、社交账户或投资账户，而不只是功能账户。

入脑表达，是 30 秒能听懂、能复述、愿意付费的一句话。

### 品类通关检查

```text
1. 用十六字真言验证品类，四维全部通过。
2. 锁定甜用户，S级客户 ≥ 5。
3. 找到至少3个贵任务，进入情绪 / 社交 / 投资账户。
4. 30秒电梯测试通过，用户听懂且愿意复述。
5. 输出一句入脑价值主张，不超过20字。
```

## 02.6 第三关：模式升维

很多创业项目不是没人买，而是不能复制。

创始人很能干，客户很满意，项目很热闹，但一离开创始人，系统就停。

这不是商业模式。

这是高级个体户。

模式升维要解决的问题是：

```text
从个人能干，到系统能跑。
```

核心结构是三链贯通：

```text
前链路 × 后链路 × 财链路
```

前链路解决渴望度：用户为什么想要？怎么找到他？怎么让他愿意进入？

后链路解决可行性：你凭什么稳定交付？核心资源、关键业务、伙伴网络是否撑得住？

财链路解决生存性：收入来源、定价机制、成本结构是否成立？

这三条链路合起来，对应商业模式九模块。

### 模式通关检查

```text
1. 商业模式九模块画布填满。
2. 9张闭环验证测试卡至少3张跑通。
3. 三链路闭环跑通：获客 → 交付 → 变现一个真实订单。
4. 找到至少1个 LTV/CAC > 3 的超级单点。
5. ABCS 段位 ≥ A，能用数字说清模式。
```

## 02.7 第四关：壁垒锁定

赚到钱，不等于创业成功。

很多项目真正危险的时刻，不是没赚到钱，而是刚赚到钱。

因为一旦证明你能赚钱，模仿者、平台、巨头、同行、上下游都会靠近。没有壁垒，利润不是你的，只是暂时经过你。

壁垒锁定要回答的问题是：

```text
模式升维赚到了钱，怎么守住不被收割？
```

核心结构是：

```text
四维护城 × 16 战略控制点
虚 × 实 × 入 × 出
```

虚壁垒，是用户认你。品牌、心智、品类、文化叙事，让别人进不来。

实壁垒，是你做得到。极致成本、资产密度、链主控制、工艺黑箱，让别人追不上。

入壁垒，是用户离不开。学习成本、数据沉淀、生态迭代、沉没成本，让别人拿不走。

出壁垒，是用户互锁。用户网络、平台效应、双边市场、生态飞轮，让系统自己长。

### 壁垒通关检查

```text
1. 16控制点至少8个达到高分锚点。
2. 至少6个控制点在持续投资中。
3. 客户转换成本可量化，流失率低于目标线。
4. 至少1个壁垒达到“竞品6个月追不上”。
5. 输出一页纸“我的护城河”，含证据与30法则推演。
```

## 02.8 四获得物：从顺风到飞轮

原力创业不是每关完成一项作业，而是每关带走一份可复利资产。

```text
势能地图 → 心理账户 → 利润容器 → 财富飞轮
```

白话版是：

```text
顺风 → 开户 → 容器 → 飞轮
```

原力资产是创业前置输入，不再是创业第一关输出。

第一关，势能地图是顺风。你知道自己的合力该放进哪个时代窗口。

第二关，心理账户是开户。你不再停留在功能价值，而是进入用户真正愿意为你调动预算的心智账户。

第三关，利润容器是容器。你不再靠人肉交付，而是把前链路、后链路、财链路接起来，让价值可以复制。

第四关，财富飞轮是飞轮。你不只是赚到钱，而是让利润被壁垒锁住，形成长期复利。

## 02.10 本章边界

《原力创业》只负责三件事：

```text
把原力资产放进时代窗口。
把市场需求做成利润容器。
把利润容器锁成财富飞轮。
```

它不负责重新解码“你”的全部深层结构，那是《原力资产》。

它不负责搭 AI 班底、自动化链路、知识库和工作生命体，那是《原力OS》。

它也不负责具体投资组合和家庭资产配置，那是未来《原力财富》。

本章的终点，是一句话：

> 我知道如何把我的原力，放进一股正确的时代势里，变成一门可通关、可复利、守得住的生意。

从这里开始，我们进入第三部：

> **原力OS。**
>
> 把做事系统，变成会自我繁殖的工作生命体。
