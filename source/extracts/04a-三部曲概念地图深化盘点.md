## 1. 总体分层：从母公理到页面外显

### L0 · 母公理

```text
ʸx：做会自我繁殖的事，不做一次性的事。
```

判断标准：

```text
完成一次之后，是否让下一次更容易？
完成一个之后，是否能孵化下一个？
完成一轮之后，是否能回写成资产、能力、系统或判断？
```

### L1 · 公理层 umbrella

```text
原力战略白皮书 = 公理层 umbrella
```

白皮书坐在三本书之上，回答“为什么整套体系必然成立”，不再作为三本之一。

### L2 · 三部曲

```text
原力战略三部曲 = 原力资产 / 原力创业 / 原力OS
```

边界：

```text
原力资产：向内 / 人 / 找到你
原力创业：向外 / 事 / 变现你
原力OS：向上 / 法 / 系统你
```

### L3 · 三大母公式

```text
原力资产 = 「你」 × AI
财富 = 借势 × 合力
原力OS = 文脉 × 大脑 × 地图 × 链路 × 递归回写
```

### L4 · 共享脊骨


## 2. 第一部：原力资产 · 3-6 级概念结构

### L2 · 定义

```text
原力资产 = 「你」 × AI = 会自我繁殖、万倍放大的资产。
```

本质不是兴趣探索，也不是优势分析，而是身份级资产确权。

### L3 · 核心方法论：原力U

```text
觉察 → 高能 → 阴影 → 整合 → 资产 → 复利
```

纵深逻辑：

```text
从可见症状进入结构，
从结构潜到阴影源头，
从源头整合上升，
最终浮出为可被 AI 放大的资产。
```

```text
1. 「你」× AI = 原力资产
```

## 3. 第二部：原力创业 · 3-6 级概念结构

### L2 · 定义

```text
原力创业，是拿着已经确认的原力资产，去寻找它最该搭上的时代顺风，并把它变成可通关、可复利、守得住的生意。
```

短公式：

```text
原力创业 = 借势 × 合力 × 四关通关
```

### L3 · 第一性公式

```text
财富 = 借势 × 合力
```

其中：

```text
合力 = 内部不变量 = 原型力 / 三圈交集 / 原力资产
借势 = 外部不变量 = 康波势 / 技术势 / 产业势 / 结构势 / 用户心智势
```

### L4 · 新四关

```text
原力借势 → 品类独创 → 模式升维 → 壁垒锁定
```

四关获得物：

```text
势能地图 → 心理账户 → 利润容器 → 财富飞轮
```

### L5 · 第一关：原力借势

```text
原力借势 = 宏周期势 × 中结构势 × 微入口势 × 合力匹配
```

四层：

```text
宏周期势：康波 / 技术范式 / 资本周期 / AI渗透率
中结构势：政策 / 经济 / 社会 / 技术 / 产业价值链 / PESTEL
微入口势：用户心智 / 心理账户 / 渠道窗口 / 预算迁移
合力匹配：原力资产是否刚好适配这股势
```

输出：

```text
势能地图
原力借势卡
PEST势能扫描卡
康波窗口判断卡
机会假设卡
初始进入点
```

通关检查：

```text
1. 能 30 秒说清：我借的是哪一股势。
2. 能用康波 / 技术 / 产业 / PEST 至少三类证据证明这股势存在。
3. 能说明：为什么是现在，不是三年前，也不是三年后。
4. 能证明：我的原力资产和这股势同向，而不是硬蹭风口。
5. 能输出明确进入点：甜用户、贵任务、预算来源、品类机会。
```

### L5 · 第二关：品类独创

```text
目标：不在旧品类中竞争，而是创造无法分类、重新定义、自带预算的新位置。
十六字真言：无法分类 / 重新定义 / 自带预算 / 品类独创
落地公式：三角定位 = 甜用户 × 贵任务 × 入脑表达
心理账户：功能 / 情绪 / 社交 / 投资
输出：心理账户 / 独创品类 / 一句入脑 / MVP闭环
```

### L5 · 第三关：模式升维

```text
目标：从个人能干，到系统能跑。
核心公式：三链贯通 = 前链路 × 后链路 × 财链路
九模块：用户细分 / 价值主张 / 渠道结构 / 客户关系 / 核心资源 / 关键业务 / 伙伴网络 / 收入来源 / 定价机制 / 成本结构
输出：利润容器 / 三链路闭环 / 可复制模式
```

### L5 · 第四关：壁垒锁定

```text
目标：模式升维赚到了钱，怎么守住不被收割。
核心结构：四维护城 × 16 战略控制点 = 虚 × 实 × 入 × 出
目标状态：进不来 / 追不上 / 拿不走 / 离不开
输出：财富飞轮 / 护城河 / 穿越周期能力
```

## 4. 第三部：原力OS · 3-6 级概念结构

### L2 · 定义

```text
原力OS，是一个由文脉、大脑、地图、链路组成的工作生命体；它在三法印约束下，把每一次任务递归回写成可复用能力，最终让人从 AI 使用者进化为 ʸx 孵化者。
```

公式：

```text
原力OS = 文脉 × 大脑 × 地图 × 链路 × 递归回写
```

白话：

```text
懂你 + 记得住 + 会判断 + 能动手 + 每次变强
```

### L3 · 数学地基

```text
1·x   你 + 1 个工具
x·y   你 + N 个工具
x^y   你 + N 个工具 + 工具自反馈
ʸx    你 + N 个智能 + 智能造下一代智能
```

关键：

```text
x^y 仍然需要人在场推动。
ʸx 的关键是：你不在场时，系统仍在生长。
```

### L4 · 本体层：四组件

```text
文脉 = 灵魂 = 我是谁
大脑 = 记忆 = 我知道什么
地图 = 判断 = 我怎么决策
链路 = 手脚 = 我怎么行动
```

完备性：

```text
缺文脉：AI 不懂你，只是通用 AI。
缺大脑：系统失忆，每次从零。
缺地图：系统会瞎干，缺判断。
缺链路：只想不做，不能行动。
```

### L4 · 宪法层：三法印

```text
递归进化：每件事比上件好。
技能统帅：先 search 再 call。
人类友好：节省 token、不打扰、保留人的最终判断权。
```

优先级：

```text
人类友好 > 递归进化 > 技能统帅
```

### L4 · 运行层：递归回写

```text
任务 → 输出 → 复盘 → 模式 → 技能 → 回写四组件 → 下一次任务更强
```

回写位置：

```text
新洞察 / 新边界 → 文脉
新知识 / 新案例 → 大脑
新因果 / 新判断 → 地图
新 SOP / Prompt / Agent → 链路
可复用能力 → skill 库
```

### L4 · 任务层：OSA

```text
O 目标层：我们要干什么？为什么是这件不是那件？
S 策略层：怎么打？哪条路径成功率最高？候选 ABC 是什么？
A 行动层：谁做？何时做？怎么验真？
```

### L4 · 判断层：信息五层

```text
碎片 → 特征 → 观点 → 洞察 → 框架
```

### L4 · 组织层：AI 班底

```text
1 = 你，碳基启动者
N = 你的 AI 班底
+N 自繁殖 = N 能造下一代 N
```

梯度：

```text
1 + 3   入门
1 + 10  合格
1 + 30  优秀
1 + 100 S4
```

### L4 · 资产层：case → pattern → skill

```text
case：一次性事件
pattern：重复出现的模式
skill：可被反复调用的能力
```

版本：

```text
v1 雏形
v2 跑通
v3 通用化
v4 进库
```

### L4 · 成熟层：S路径

```text
S1 建模 = 1·x = 学生
S2 工具 = x·y = 工匠
S3 板   = x^y = 产品人
S4 舱   = ʸx  = 孵化者
```

### L6 · 实操模板

```text
文脉卡
大脑库卡
决策地图卡
链路卡
Evolution Note
Skill蒸馏卡
```

### 边界

```text
OS 不负责重新找到你，那是原力资产。
OS 不负责判断借哪股势、做哪门生意，那是原力创业。
OS 负责把资产与创业产生的经验，变成可复用、可调用、可进化的系统。
```
