## 0. 总裁决

这份 30 节课源稿在三部曲新正典中的归属是：

```text
主归属：《原力资产》
副接口：《原力创业》
副接口：《原力OS》
潜在接口：未来《原力财富》
```

核心原因：

```text
它的主任务不是教创业，也不是教工具，而是完成“原力觉醒”：
让 AI 理解你、学习你、放大你，最终生成原力资产。
```

但其中 L21-L25 明显进入“资产 → 创业”的接口，L26-L30 又进入“资产 → 财富 / 借势 / 确定键”的出口。因此不能简单全部塞进《原力资产》正文，而应该作为《原力资产》的课程型源头与接口桥。

## 2. 课程主线萃取

这套课的主线不是“教 AI”，而是：

```text
算力认输 → 元AI反转 → 自我解码 → 上下文喂养 → 能量整合 → 用户显形 → 非线性变现 → 按下确定键
```

对应《原力资产》主线：

```text
觉察 → 高能 → 阴影 → 整合 → 资产 → 复利
```

对应三部曲接口：

```text
觉醒生成合力 → 合力进入创业借势 → 创业经验交给OS回写 → OS反哺资产升级
```

### L01 · 关窍：人类文明的唯一秘密

**新正典归属：** 原力资产 / 觉察  
**核心命题：** 成功不是靠努力，而是靠找到全局最优解。  
**关键概念：** 关窍、关窍五步法、信息处理学、全局最优解、算力差距。  
**反认知：** 努力不是答案，算力才是；勤奋只解决20%，关窍解决80%。  
**AI实操：** 拆解所在行业 Top3 关窍，对比策略偏差。  
**输出物：** 行业关窍扫描卡。  
**接口：** OS 的信息五层；创业的行业关窍/机会判断。
