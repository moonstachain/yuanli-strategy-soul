## 2. B1 可吸收核心判断

### 2.1 财富公式

```text
财富 = 借势 × 合力
```

解释：

```text
借势 = 外部时代周期之势
合力 = 内部人格 / 原力 / 三圈交集之力
只有两者同时在场，才有非线性回报。
```

### 2.3 康波势

源仓将“势”归约为康波周期，而不是短期风口：

```text
风口太短，政策太局部，技术接近但不够本质，产业周期只是子周期。
真正的元周期是康波周期。
```

B1 应吸收的判断：

```text
势 = 康波周期 × 产业结构 × 用户入口。
康波是最高层，产业/政策/渠道是中层，用户心智与预算迁移是微入口。
```

回写位置：

```text
glossary/startup-创业术语.md
templates/startup/原力借势卡.md
```

### 2.4 K6 千日窗口

关键判断：

```text
K5 信息互联网进入深冬；K6 AI / 生物 / 能源进入初春。
2025-2028 是第六康波的千日窗口。
窗口期的核心不是确定性，而是心智占位与品类定义权。
```

### 2.5 借势两大算法

源仓给出两种最优借势算法：

```text
算法 A：旧周期做极致性价比
算法 B：新周期做创新十倍好
```

转译为 B1 工具：

```text
如果你借 K5 存量势：做极致性价比、供应链压缩、体验重构。
如果你借 K6 增量势：做创新十倍好、流程压缩、效果付费、AI 替代人力。
```
