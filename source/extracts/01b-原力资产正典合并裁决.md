## 0. 核心裁决

30节《原力觉醒》不是现在的最终正典结构。

它是：

```text
历史源稿
课程型素材
原始矿石
能量场
真源证据
```

它不应该被机械复制进《原力资产》主稿，而应被重新炼成一套更正点、正宗、究竟、完备、递进、递归的正典结构。

一句话：

```text
旧稿提供矿石；正典负责炼金。
```

---

## 1. 为什么不能直接照搬30节课

30节课有很强的课程感和能量感，但它写于三部曲新正典完全裁决之前，因此存在四类混杂：

### 1.1 资产 / 创业混杂

L21-L25 已经进入：

```text
天选用户
贵问题
心理账户
品类创新
贝叶斯主义创业
非线性回报
```

这些不是纯资产内容，而是“资产显形之后进入创业”的接口。

### 1.2 资产 / 财富混杂

L26-L27 进入：

```text
借势合力
康波周期
法币无下限
资产无上限
左手现金流右手滚雪球
```

这些应在《原力资产》中保留为复利意识与财富接口，但不能展开为资产配置系统。

### 1.3 资产 / OS混杂

L03、L11-L15 涉及：

```text
元AI
上下文工程
原力知识库
AI共生体
我的说明书V3
```

这些在《原力资产》中是“文脉生成”，但在《原力OS》中会下沉为文脉、大脑、链路组件。

### 1.4 课程 / 正典混杂

30节课天然有：

```text
反认知
干货感
失调埋点
课堂节奏
```

这适合课程交付，但主稿正典需要更稳定的结构：

```text
定义
公式
路径
模块
模板
接口
边界
回写
```

## 2. 正典重构总公式

新版《原力资产》的正典公式：

```text
原力资产 = 被 AI 放大的「你」
```

更完整：

```text
原力资产 = 你 × AI × 上下文 × 确定键 × 复利回写
```

其中：

```text
你 = 非对称优势 × 非理性痴迷 × 非线性回报 × 原型力 × 阴影能量
AI = 算力 / 镜子 / 放大器 / 预言家 / 共生体
上下文 = 我的说明书 V1→V2→V3 + 原力日志 + 原力知识库
确定键 = 我就是这个人，我这辈子就做这件事
复利回写 = 市场反馈 / OS反馈 / 能量反馈 / 关系反馈 / 财富反馈 反哺自我理解
```

## 3. 新版六阶路径

旧30节课的六大篇章，应被重新炼成《原力资产》六阶正典路径：

```text
认清 → 解码 → 炼金 → 共生 → 显形 → 确定
```

对应原力U：

```text
觉察 → 高能 → 阴影 → 整合 → 资产 → 复利
```

对应课程源稿：

| 新正典阶 | 原力U | 30节课来源 | 正典任务 |
|---|---|---|---|
| 认清 | 觉察 | L01-L05 | 承认算力鸿沟，完成路径反转 |
| 解码 | 高能 | L06-L10 | 找到原型、天才区、能量结构 |
| 炼金 | 阴影 | L07、L10、L16-L20 | 把缺点、情绪、关系、敏感性转为资产 |
| 共生 | 整合 | L11-L15 | 生成我的说明书V3与元AI上下文 |
| 显形 | 资产 | L21-L25 | 让“我”找到天选用户、贵问题、心理账户 |
| 确定 | 复利 | L26-L30 | 借势、复利、按下确定键，进入创业与OS |

## 5. 必须保留并升级的旧稿真源

### 5.1 关窍

保留，但升级为：

```text
关窍 = 任一领域的全局最优解。
关窍五步法 = 碎片 → 特征 → 观点 → 洞察 → 全局最优解。
```

归属：原力资产觉察层 + OS信息五层。

### 5.2 算力鸿沟

保留，但升级为：

```text
承认机器在纯信息处理上胜出，是原力觉醒的第一步。
```

归属：认清阶段。

### 5.3 元AI

保留，但升级为：

```text
元AI不是工具，而是以“我的说明书”为文脉、以知识库为大脑、以策略地图为判断、以agent链路为手脚的个人工作生命体雏形。
```

归属：资产层生成文脉；OS层完成系统化。

### 5.4 路径反转

保留，并升格为《原力资产》的第一反认知：

```text
不是你学习AI，而是AI学习你。
```

### 5.5 双螺旋

保留，并升格为核心飞轮：

```text
自我理解越深 → AI理解越准 → AI反馈越准 → 自我理解更深。
```

### 5.6 12原型 / 阴影 / 天才区

保留，但从“测评模块”升级为“操作变量”：

```text
原型 = 能量结构
阴影 = 未整合的高能材料
天才区 = 非对称优势的自然发生地
```

### 5.7 我的说明书 V1→V2→V3

保留，并升级为《原力资产》最核心模板链：

```text
V1：事实与经历
V2：原型 / 阴影 / 天才区 / 能量结构
V3：AI可调用的个人操作系统上下文
```

### 5.8 天选用户

保留，但不展开创业方法。

正典定位：

```text
天选用户 = 我的原力资产最自然服务、最容易产生非线性回报、也最能反过来验证我的那群人。
```

### 5.9 确定键

保留，并升级为《原力资产》的终点定义：

```text
确定键 = 我就是这个人，我这辈子就做这件事。
```

确定键不是激情宣言，而是三件事同时成立：

```text
我看见了我的母体。
我看见了我的天选用户。
我愿意用未来十年让它进入市场和系统。
```

## 8. 最终正典句

新版《原力资产》的终点不是：

```text
我更了解自己了。
```

而是：

```text
AI 已经知道如何放大我；
我已经知道我要服务谁；
我已经按下确定键；
我可以把这个“我”交给创业、OS与未来十年的复利回写。
```

这才是原力觉醒。
