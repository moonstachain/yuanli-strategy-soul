## 1. 本轮裁决

《原力创业》的第一关不再是“原力觉醒”，而是“原力借势”。

```text
原力觉醒：归入《原力资产》，负责生成合力。
原力借势：归入《原力创业》，负责让合力进入时代窗口。
```

因此，“借势”必须从一个宏观词变成一个可操作系统：

```text
原力借势 = 宏周期势 × 中结构势 × 微入口势 × 合力匹配
```

---

## 2. 借势的第一性：财富 = 借势 × 合力

`books/01-道-dao.md` 把第一册定位为“借势×合力第一性”，明确本册只解决一件事：把“财富 = 借势 × 合力”立为整门生意的地基公式。

该源头通过三个观察推出财富不是单一变量：

```text
同样的能力，在不同时代，财富差 1000 倍。
同样的时代，不同的人，财富差 1000 倍。
同样时代+能力，做不同的事，财富差 100 倍。
```

继续追问后，归到两个不变量：

```text
外部时代有周期 = 借势。
内部人格有底色 = 合力。
```

关键结论：

```text
财富 = 借势 × 合力
借势 = 康波势
合力 = 原型力
```

这给“原力借势”一个明确边界：

```text
它不是让创始人追热点，而是先判断外部时代周期是否正在放大自己的合力。
```

---

## 3. 宏周期势：康波是元周期，不是风口

源头中明确区分：

| 候选 | 尺度 | 判定 |
|---|---|---|
| 风口 | 6–18个月 | 太短，纯炒作 |
| 政策 | 局部 | 太局部 |
| 技术 | 中等 | 接近但不够本质 |
| 产业周期 | 数年 | 康波的子周期 |
| 康波周期 | 50–60年 | 真正不变的元周期 |

因此，《原力创业》的第一关必须先用“尺度”淘汰假势：

```text
风口不是势。
政策不是势。
单一技术不是势。
产业周期也不是最终势。
它们都只是康波的切片、局部或子周期。
```

新的判断口径：

```text
宏周期势 = 先判断你站在哪一轮康波、哪个季节、哪个窗口。
```

## 4. K1-K6 与千日窗口

源头给出 K1-K6 速查：

```text
K1 1780–1840：蒸汽机 / 纺织
K2 1840–1890：铁路 / 钢铁 / 电报
K3 1890–1940：电力 / 内燃机 / 化学
K4 1940–1990：汽车 / 石油 / 计算机萌芽
K5 1990–2026：PC / 互联网 / 移动 / 社交
K6 2025–2080：AI / 生物 / 能源 / 脑机
```

创业绿皮书和通关地图共同给出时代锚点：

```text
第五次康波深冬 × 第六次康波初春
2025-2028 = 第六次康波的千日窗口
```

原力借势的第一判断：

```text
我是在守 K5 存量，还是在借 K6 增量？
```

进一步判断：

```text
我所使用的核心技术，处在能力指数上升期，还是平台期？
我所在产业处于萌芽、成长、成熟，还是衰退？
我现在入场，是早、刚好，还是晚？
```

## 5. 技术范式势：第六康波为什么不同

OS 绿皮书第02章强调：过去五次康波本质上都是工具升级，放大的是碳基人的效率；但第六次康波的不同在于杠杆载体变了。

```text
第1-5次康波：工具升级，放大碳基效率。
第6次康波：杠杆载体从碳基切换到硅基。
```

这给“技术范式势”一个更深判断：

```text
不是“我用了 AI”，而是这个行业是否因为 AI 改变了杠杆载体。
```

判断标准：

```text
AI 是否把原来需要人的关键流程变成可自动化、可复制、可扩展？
AI 是否让旧行业的 10 步变成 1 步？
AI 是否让按过程收费变成按效果收费？
AI 是否让原本不可规模化的专家能力被产品化？
```

## 6. 中结构势：PEST / PESTEL 是康波的切片扫描

PEST 不应取代康波，而应作为中观扫描工具。

关系如下：

```text
康波 = 元周期
PEST / PESTEL = 元周期落到现实世界的结构切片
产业链 = 元周期在具体行业中的价值迁移路径
```

PEST 四问：

```text
P 政策：哪些政策、监管、公共议题正在改变供需？
E 经济：哪些收入、成本、资本、价格机制正在变化？
S 社会：哪些人群、代际、身份、生活方式正在迁移？
T 技术：哪些技术曲线正在进入可用、便宜、普及阶段？
```

PESTEL 可扩展：

```text
E 环境：资源、能源、气候、可持续约束是否改变成本结构？
L 法律：合规、数据、隐私、版权、劳动关系是否改变行业边界？
```

关键裁决：

```text
PEST 不是写报告，是判断“哪一个结构性变化正在替我的原力资产打开一扇门”。
```

## 7. 产业势：找价值迁移，不找抽象赛道

产业势不是“这个行业热不热”，而是：

```text
价值正在从哪里迁到哪里？
谁的利润池正在被压缩？
谁的成本结构正在被 AI 改写？
谁的中间环节正在被砍掉？
谁的用户心智正在松动？
```

两条借势算法：

```text
旧周期算法：极致性价比。
新周期算法：创新十倍好。
```

旧周期极致性价比：

```text
当一个行业创新已结束，整合前人创新，品质足够，价格良心者会拿走市场。
```

新周期创新十倍好：

```text
用第一性原理和新技术，把暴利行业或低效流程重新做一遍，做到 10 倍好。
```

这给原力借势一个行业选择标准：

```text
K5 存量行业：找可整合、可降本、可脚踝斩的机会。
K6 增量行业：找 AI 可以把 10 步压成 1 步、把专家能力产品化的机会。
```

## 8. 微入口势：用户心智与预算迁移

宏周期和中结构还不能直接变成生意，必须落到微入口：

```text
谁最先感到变化？
谁的旧方案正在失效？
谁的预算正在迁移？
谁的心理账户已经被打开？
谁愿意为新解决方案付第一笔钱？
```

微入口势连接第二关“品类独创”：

```text
原力借势输出初始进入点。
品类独创把初始进入点变成心理账户和一句入脑表达。
```

微入口的四个判断：

```text
用户心智是否松动？
旧品类是否解释不了新问题？
预算是否从旧方案迁移到新任务？
渠道或内容窗口是否刚刚打开？
```

---

## 9. 合力匹配：不是硬蹭风口

源头反复强调：势对力不对，乘积依然趋近于 0。

原力借势最后一层必须回到《原力资产》：

```text
我的原力资产是否刚好适配这股势？
我的非对称优势是否能在这股势里被放大？
我的非理性痴迷是否能支撑长期投入？
我的非线性回报是否能被 AI / 市场 / 系统放大？
我的母体定义卡是否能自然长出这个机会？
```

结论：

```text
不是所有大势都是我的势。
只有能放大我这颗母体的大势，才叫原力借势。
```

## 10. 原力借势的标准流程

```text
第一步：确认合力输入
  读取母体定义卡、三圈交集、天才区、原型力。

第二步：扫描宏周期势
  判断 K5/K6、康波季节、千日窗口、技术范式。

第三步：扫描中结构势
  用 PEST / PESTEL 和产业链图找结构变化。

第四步：扫描微入口势
  找用户心智松动、心理账户打开、预算迁移、渠道窗口。

第五步：做合力匹配
  判断这股势是否真的适合我，而不是硬蹭。

第六步：输出机会假设
  写出甜用户、贵任务、预算来源、品类机会、MVP入口。
```

## 11. 第一关通关标准 v0.2

```text
1. 能 30 秒说清：我借的是哪一股势。
2. 能用康波 / 技术 / 产业 / PEST 至少三类证据证明这股势存在。
3. 能说明：为什么是现在，不是三年前，也不是三年后。
4. 能证明：我的原力资产和这股势同向，而不是硬蹭风口。
5. 能输出明确进入点：甜用户、贵任务、预算来源、品类机会。
```

## 14. 本轮结论

原力借势不是：

```text
追风口
看政策
押热点
套PEST模板
```

而是：

```text
拿着已经确认的原力资产，
用康波判断大时间，
用PEST扫描中结构，
用用户心智和预算迁移寻找小入口，
最后验证这股势是否真的能放大我的合力。
```

一句话：

```text
原力借势 = 找到“时代正在替我推门”的那个入口。
```
