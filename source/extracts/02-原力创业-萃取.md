## 1. 一句话定义

```text
原力创业，是把“你”这颗原力资产，放进时代、用户、现金流与壁垒系统里，最终形成可通关、可复利、守得住的生意。
```

更短版本：

```text
原力创业 = 借势 × 合力 × 四关通关
```

它不是普通创业学，也不是 AI 创业指南，而是一张通关地图：

```text
一个公式：财富 = 借势 × 合力
四个关口：原力觉醒 → 品类独创 → 模式升维 → 壁垒锁定
一句话目标：左手现金流，右手资产雪球
```

## 2. 第一性公式：财富 = 借势 × 合力

原力创业的核心公式：

```text
财富 = 借势 × 合力
```

其中：

```text
借势 = 外部不变量 = 康波势 / 技术势 / 产业势 / 个人周期势
合力 = 内部不变量 = 原型力 / 三圈交集 / 原力资产
```

为什么是乘法：

```text
有合力但无势：怀才不遇，只能线性挣扎。
有势但无合力：吃不到属于你的红利，容易成为风口炮灰。
任一为 0，整体归零。
```

人话版：

```text
做自己 = 合力。
好生意 = 借势 × 把合力翻译成钱。
```

边界：合力深处归《原力资产》，创业只调用资产输出，并把它推进市场、模式与壁垒。

## 3. 时代锚点：千日窗口

原力创业不是从零造势，而是识别已有的势。

当前关键判断：

```text
第五次康波（信息技术）深冬 × 第六次康波（AI / 生物科技 / 能源）初春
```

每次范式转移的前 1000 天，是建立品类定义权与用户心智的关键窗口。

```text
2025-2028 = 第六次康波的千日窗口
```

原力创业的任务，是让创始人不错过这个窗口：

```text
不从零造势，先识别势。
不靠蛮力硬卷，先让母体顺风繁殖。
```

## 4. 核心结构：四关

```text
原力觉醒 → 品类独创 → 模式升维 → 壁垒锁定
```

四关不是目录，而是商业闭环的四个生死端口。

| 关 | 方向 | 第一性问题 | 获得物 | 失败形态 |
|---|---|---|---|---|
| 原力觉醒 | 向内求 | 在 AI 能替代标准化能力的时代，你唯一不可替代的资产是什么？ | 原力资产 / 能量原点 | 卖身、内耗、转行 |
| 品类独创 | 向外立 | 找到原力后，怎么把它变成市场认得的钱？ | 心理账户 / 预算锁定 | 同质化、价格战 |
| 模式升维 | 向上升 | 找到独创品类后，怎么把它做成可复制的赚钱机器？ | 利润容器 / 模式空间释放 | 有客无利、人肉换钱 |
| 壁垒锁定 | 向守远 | 赚到了钱，怎么守住不被巨头收割？ | 财富飞轮 / 穿越周期 | 被抄走、利润归零 |

## 5. 第一关：原力觉醒

### 5.1 关口定义

```text
原力觉醒 = 非对称优势 ∩ 非理性痴迷 ∩ 非线性回报
```

三圈必须同时成立：

| 圈 | 维度 | 解释 | 测试 |
|---|---|---|---|
| 非对称优势 | 能力 / 输入 | 你独有的、别人无法模仿的特质 | 5维判断 |
| 非理性痴迷 | 动机 / 过程 | 即使不赚钱也忍不住要做 | 6信号 |
| 非线性回报 | 结果 / 输出 | 扩100倍不需要线性增加精力 | 5信号 + 杠杆4层 |

### 5.2 关口输出

```text
一句话天赋定位
原力资产
能量原点
```

### 5.3 通关检查

```text
1. 三圈交集 30 秒说清。
2. 完成 12 原型测评，主/次原型清晰。
3. 天才区能量回流测试完成，知道天才区在哪一象限。
4. 3-2-1 阴影练习完成，金色阴影方向清晰。
5. 输出一句话天赋定位，不超过 30 字。
```

## 6. 第二关：品类独创

### 6.1 关口定义

品类独创不在现有品类里竞争，而是创造一个无法分类、重新定义、自带预算的新品类。

十六字真言：

```text
无法分类 / 重新定义 / 自带预算 / 品类独创
```

落地公式：

```text
三角定位 = 甜用户 × 贵任务 × 入脑表达
```

### 6.2 三角定位

| 角 | 问题 | 判断标准 |
|---|---|---|
| 甜用户 | Who | 你最强禀赋能服务、且愿意长期为你买单的人群 |
| 贵任务 | What | 落在情绪 / 社交 / 投资账户，而不只是功能账户 |
| 入脑表达 | How | 30秒听懂，能复述，愿付费 |

### 6.3 四大心理账户

```text
功能账户：有上限，容易卷价格。
情绪账户：为被看见、被安放、被疗愈付费。
社交账户：为身份、圈层、面子、归属付费。
投资账户：为未来收益、能力增值、机会放大付费。
```

品类独创的关键，是把原力资产从功能账户里拔出来，进入更高阶心理账户。

### 6.4 通关检查

```text
1. 16字真言四维全通过。
2. 甜用户4维筛选通过，S级客户 ≥ 5。
3. 贵任务进入情绪 / 社交 / 投资账户。
4. 30秒电梯演讲通过，一句话品类定位清晰。
5. MVP闭环跑通，出现真实转化。
```

## 7. 第三关：模式升维

### 7.1 关口定义

模式升维解决的问题：

```text
从“个人能干”到“系统能跑”。
```

核心公式：

```text
三链贯通 = 前链路 × 后链路 × 财链路
```

对应九模块：

```text
前链路：用户细分 / 价值主张 / 渠道结构 / 客户关系
后链路：核心资源 / 关键业务 / 伙伴网络
财链路：收入来源 / 定价机制 / 成本结构
```

### 7.2 58 商业模式原子

模式不是凭空发明，而是组合。

```text
人类商业 300 年验证过的基本组件，可以压缩为 58 种商业模式原子。
```

创业者要做的，不是发明新商业，而是在自己的康波、用户、品类和能力约束下，找到一组可跑通的原子组合。

### 7.3 通关检查

```text
1. 商业模式九模块画布填满。
2. 9张闭环验证测试卡至少3张跑通。
3. 三链路闭环跑通：获客 → 交付 → 变现一个真实订单。
4. 找到至少1个 LTV/CAC > 3 的超级单点。
5. ABCS 段位 ≥ A，能用数字说清模式。
```

## 8. 第四关：壁垒锁定

### 8.1 关口定义

壁垒锁定解决的问题：

```text
模式升维赚到了钱，怎么守住不被收割？
```

核心结构：

```text
四维护城 × 16 战略控制点
虚 × 实 × 入 × 出
```

目标：

```text
进不来 / 追不上 / 拿不走 / 离不开
```

### 8.2 四维护城

| 壁垒 | 作用 | 控制点方向 |
|---|---|---|
| 虚 · 心智 | 用户认你，进不来 | 品牌溢价 / 特许 / 品类 / 文化叙事 |
| 实 · 物理 | 你做得到，追不上 | 极致成本 / 资产密度 / 链主 / 工艺黑箱 |
| 入 · 锁定 | 用户离不开，拿不走 | 学习成本 / 数据沉淀 / 生态 / 沉没成本 |
| 出 · 网络 | 用户互锁，离不开 | 用户网络 / 平台 / 双边 / 飞轮 |

说明：早期文档中也出现过 VRSE 的“虚/实/出/入”顺序，本轮采用三部曲重构后的“虚/实/入/出”四维护城口径；后续需要建立术语对照表，避免外显时混乱。

### 8.3 通关检查

```text
1. 16控制点至少8个达到高分锚点。
2. 至少6个控制点在持续投资中。
3. 客户转换成本可量化，流失率低于目标线。
4. 至少1个壁垒达到“竞品6个月追不上”。
5. 输出一页纸“我的护城河”，含证据与30法则推演。
```

## 9. 四获得物：创业资产生命线

四关不是只为了通关，而是每一关带走一份可复利资产：

```text
原力资产 → 心理账户 → 利润容器 → 财富飞轮
```

白话：

```text
本金 → 开户 → 容器 → 飞轮
```

| 关 | 获得物 | 一句话 |
|---|---|---|
| 觉醒 | 原力资产 | 本金：找到我 |
| 独创 | 心理账户 | 开户：说清我 |
| 升维 | 利润容器 | 容器：放大我 |
| 锁定 | 财富飞轮 | 飞轮：焊死我 |

合体：

```text
左手现金流 = 心理账户 × 利润容器的当期产出
右手资产雪球 = 财富飞轮的长期复利
出师 = 左手现金流 + 右手资产雪球
```

## 10. 通关纪律

原力创业最重要的纪律：不要跳关。

```text
觉醒没完成，不要做品类独创。
品类没跑通，不要做模式升维。
模式没闭环，不要做壁垒锁定。
```

为什么不能跳关：

| 跳过 | 后果 |
|---|---|
| 跳过觉醒 | 没母体，做着做着动力崩溃 |
| 跳过独创 | 在已有品类卷价格，毛利被打穿 |
| 跳过升维 | 没现金流，壁垒投入打水漂 |
| 跳过锁定 | 利润被抄走，为他人作嫁衣 |

全部通关：

```text
出师 = 触发下一阶段投资机会 + 原力人生融合
```

## 11. 与传统创业方法论的差异

| 传统创业方法 | 原力创业 |
|---|---|
| 从零造势 | 借势：识别康波 / 技术 / 产业 / 个人周期 |
| 找痛点 | 合力：原型力 × 三圈交集 |
| idea → MVP → PMF → Scale | 四关严格递进，每关5项硬出师条件 |
| Pivot when broken | 品类独创：一开始就创造无法分类的新位置 |
| Hustle / grind | 借势 ≥ 努力，势对力对才有非线性回报 |
| 商业模式发明 | 58商业基因 × 九模块 × 三链路组合 |
| 护城河 = 资本 / 网络效应 | 四维护城 × 16控制点 |

## 15. 本轮结论

《原力创业》的中心不是“怎么创业”，而是：

```text
如何把原力资产放进一个会放大的时代势能里，沿四关变成可复利、守得住的生意。
```

它的最终产物不是一份商业计划书，而是：

```text
原力资产 → 心理账户 → 利润容器 → 财富飞轮
```

通完四关，才叫出师。
