## 2. 一句话定义

```text
原力创业，是拿着已经确认的原力资产，去寻找它最该搭上的时代顺风，并把它变成可通关、可复利、守得住的生意。
```

更短版本：

```text
原力创业 = 借势 × 合力 × 四关通关
```

它不是普通创业学，也不是 AI 创业指南，而是一张通关地图：

```text
一个公式：财富 = 借势 × 合力
四个关口：原力借势 → 品类独创 → 模式升维 → 壁垒锁定
一句话目标：左手现金流，右手资产雪球
```

---

## 3. 第一性公式：财富 = 借势 × 合力

原力创业的核心公式：

```text
财富 = 借势 × 合力
```

其中：

```text
合力 = 内部不变量 = 原型力 / 三圈交集 / 原力资产
借势 = 外部不变量 = 康波势 / 技术势 / 产业势 / 结构势 / 用户心智势
```

为什么是乘法：

```text
有合力但无势：怀才不遇，只能线性挣扎。
有势但无合力：吃不到属于你的红利，容易成为风口炮灰。
任一为 0，整体归零。
```

人话版：

```text
做自己 = 合力。
好生意 = 借势 × 把合力翻译成钱。
```

边界：合力深处归《原力资产》，创业只调用资产输出，并先完成“势”的判断。

## 4. 新核心结构：四关

```text
原力借势 → 品类独创 → 模式升维 → 壁垒锁定
```

四关不是目录，而是商业闭环的四个生死端口。

| 关 | 方向 | 第一性问题 | 获得物 | 失败形态 |
|---|---|---|---|---|
| 原力借势 | 向外开局 | 这个时代为什么现在需要我？ | 势能地图 | 硬蹭风口、错过窗口、怀才不遇 |
| 品类独创 | 向外立 | 找到势后，市场如何认得我？ | 心理账户 | 同质化、价格战 |
| 模式升维 | 向上升 | 找到独创品类后，怎么做成可复制的赚钱机器？ | 利润容器 | 有客无利、人肉换钱 |
| 壁垒锁定 | 向守远 | 赚到了钱，怎么守住不被收割？ | 财富飞轮 | 被抄走、利润归零 |

## 5. 第一关：原力借势

### 5.1 关口定义

```text
原力借势 = 拿着已经确认的原力资产，去寻找它最该搭上的时代顺风。
```

内部结构：

```text
原力借势 = 宏周期势 × 中结构势 × 微入口势 × 合力匹配
```

| 层级 | 看什么 | 工具 |
|---|---|---|
| 宏周期势 | 康波、技术范式、资本周期、AI渗透率 | 康波图、技术S曲线 |
| 中结构势 | 政策、经济、社会、技术、产业价值链 | PEST / PESTEL / 产业链图 |
| 微入口势 | 用户心智、心理账户、渠道窗口、预算迁移 | JTBD、心理账户、甜用户 |
| 合力匹配 | 原力资产是否刚好适配这股势 | 三圈交集、母体定义卡 |

### 5.2 关口输出

```text
势能地图
借势合力卡
千日窗口判断
机会假设卡
初始进入点
```

### 5.3 通关检查

```text
1. 能 30 秒说清：我借的是哪一股势。
2. 能用康波 / 技术 / 产业 / PEST 至少三类证据证明这股势存在。
3. 能说明：为什么是现在，不是三年前，也不是三年后。
4. 能证明：我的原力资产和这股势同向，而不是硬蹭风口。
5. 能输出明确进入点：甜用户、贵任务、预算来源、品类机会。
```

## 6. 第二关：品类独创

品类独创不在现有品类里竞争，而是创造一个无法分类、重新定义、自带预算的新品类。

十六字真言：

```text
无法分类 / 重新定义 / 自带预算 / 品类独创
```

落地公式：

```text
三角定位 = 甜用户 × 贵任务 × 入脑表达
```

四大心理账户：

```text
功能账户：有上限，容易卷价格。
情绪账户：为被看见、被安放、被疗愈付费。
社交账户：为身份、圈层、面子、归属付费。
投资账户：为未来收益、能力增值、机会放大付费。
```

通关检查：

```text
1. 16字真言四维全通过。
2. 甜用户4维筛选通过，S级客户 ≥ 5。
3. 贵任务进入情绪 / 社交 / 投资账户。
4. 30秒电梯演讲通过，一句话品类定位清晰。
5. MVP闭环跑通，出现真实转化。
```

## 7. 第三关：模式升维

模式升维解决的问题：

```text
从“个人能干”到“系统能跑”。
```

核心公式：

```text
三链贯通 = 前链路 × 后链路 × 财链路
```

对应九模块：

```text
前链路：用户细分 / 价值主张 / 渠道结构 / 客户关系
后链路：核心资源 / 关键业务 / 伙伴网络
财链路：收入来源 / 定价机制 / 成本结构
```

通关检查：

```text
1. 商业模式九模块画布填满。
2. 9张闭环验证测试卡至少3张跑通。
3. 三链路闭环跑通：获客 → 交付 → 变现一个真实订单。
4. 找到至少1个 LTV/CAC > 3 的超级单点。
5. ABCS 段位 ≥ A，能用数字说清模式。
```

## 8. 第四关：壁垒锁定

壁垒锁定解决的问题：

```text
模式升维赚到了钱，怎么守住不被收割？
```

核心结构：

```text
四维护城 × 16 战略控制点
虚 × 实 × 入 × 出
```

目标：

```text
进不来 / 追不上 / 拿不走 / 离不开
```

通关检查：

```text
1. 16控制点至少8个达到高分锚点。
2. 至少6个控制点在持续投资中。
3. 客户转换成本可量化，流失率低于目标线。
4. 至少1个壁垒达到“竞品6个月追不上”。
5. 输出一页纸“我的护城河”，含证据与30法则推演。
```

## 9. 四获得物：创业资产生命线

四关不是只为了通关，而是每一关带走一份可复利资产：

```text
势能地图 → 心理账户 → 利润容器 → 财富飞轮
```

白话：

```text
顺风 → 开户 → 容器 → 飞轮
```

| 关 | 获得物 | 一句话 |
|---|---|---|
| 借势 | 势能地图 | 顺风：找到时代窗口 |
| 独创 | 心理账户 | 开户：说清市场为什么认 |
| 升维 | 利润容器 | 容器：持续赚钱 |
| 锁定 | 财富飞轮 | 飞轮：守住并复利 |

说明：原力资产不再作为创业第一关输出，而是创业全书前置输入。

## 10. 通关纪律

原力创业最重要的纪律：不要跳关。

```text
没有借势判断，不要做品类独创。
品类没跑通，不要做模式升维。
模式没闭环，不要做壁垒锁定。
```

为什么不能跳关：

| 跳过 | 后果 |
|---|---|
| 跳过借势 | 合力找不到时代顺风，变成怀才不遇或硬蹭风口 |
| 跳过独创 | 在已有品类卷价格，毛利被打穿 |
| 跳过升维 | 没现金流，壁垒投入打水漂 |
| 跳过锁定 | 利润被抄走，为他人作嫁衣 |

## 11. 关键实操模板

### 11.1 原力借势卡

```text
我已经确认的原力资产：
我正在借的宏周期势：
我正在借的中结构势：
我正在借的微入口势：
为什么是现在：
为什么不是三年前：
为什么不是三年后：
我的合力与这股势的匹配证据：
初始进入点：
```

### 11.2 PEST 势能扫描卡

```text
P 政策：哪些政策/监管/公共议题正在改变供需？
E 经济：哪些收入、成本、资本、价格机制正在改变？
S 社会：哪些人群、代际、身份、生活方式正在迁移？
T 技术：哪些技术曲线正在进入可用、便宜、普及阶段？

结论：哪一个变化最可能成为我的顺风？
```

### 11.3 品类独创卡

```text
无法分类：我是否不容易被放进旧分类？
重新定义：我是否改写了用户对问题的理解？
自带预算：用户预算来自哪里？功能 / 情绪 / 社交 / 投资？
品类独创：我能否用一句新话定义新品类？
```

### 11.4 三角定位卡

```text
甜用户：谁最需要我，且最愿意为我付费？
贵任务：他们最贵的问题是什么？落在哪个心理账户？
入脑表达：我如何让他们 30 秒听懂，并愿意复述？
```

### 11.5 三链路画布

```text
前链路：我如何让用户渴望？
后链路：我如何稳定交付？
财链路：我如何持续赚钱？
```

### 11.6 四维护城卡

```text
虚壁垒：用户为什么认我？
实壁垒：我有什么别人追不上的真实能力？
入壁垒：用户为什么离不开我？
出壁垒：用户之间、数据之间、生态之间是否形成互锁？
```
