## 6. 吸收判断规则

任何 trilogy 内容进入 soul 前，必须判断：

```text
1. 它属于 A/B/C 哪个模块？
2. 它是正典概念、工具模板、正文素材、外显产物，还是历史参考？
3. 它是否已经在 glossary 有正名？
4. 它是否触碰去重三缝？
5. 它是否能形成输入、输出、判断标准和回写位置？
```

不能通过以上五问，不进入 soul 正典层。
