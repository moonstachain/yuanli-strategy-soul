## 2. 旧品类陷阱

The project can easily be misclassified as:

```text
场馆运营咨询
体育产业咨询
广告代理
资源中介
政府项目服务
招商服务
运营托管
```

Why this is dangerous:

```text
If users classify it as consulting or resource matching, they will compare it by proposal fee, connections, and short-term service deliverables.
That misses the real value: helping the venue's 一号位 convert dormant public assets into recurring intangible-asset cash flow.
```

---

## 3. 贵问题

The expensive problem is not venue operation.

```text
大型文体场馆一号位需要把低收益、重资产、重建设轻运营的场馆，变成可持续增值的无形资产系统。
```

Expanded expensive problem:

```text
很多大型文体场馆有空间、有流量、有政府资源，但只赚租金和基础运营收入。
一号位真正需要的是创收、政绩、标杆案例、长期运营收益和风险可控的资源整合方案。
```

Mental accounts:

```text
政绩账户
资产增值账户
风险账户
预算账户
长期现金流账户
```

## 5. 两句话品类表达

### Recommended version

```text
文体场馆无形资产开发帮助大型场馆一号位，把重建设轻运营的场馆，从租金生意升级为权益、冠名、IP和长期分润的资产增值系统。
它不同于普通场馆运营咨询，因为它不是交一份方案，而是用战略规划、资源对接和权益落地，把沉睡的公共资产变成可持续现金流。
```

### More premium version

```text
一号位资产运营工程，是面向大型文体场馆决策者的无形资产开发系统。
它解决的不是场馆有没有活动的问题，而是一座城市公共资产能不能从低收益空间，变成可创收、可汇报、可分润、可复利的长期资产。
```

### Plain-language control

```text
大型文体场馆创收和权益开发方案。
```

## 6. 用户测试问题

Blind-name test:

```text
1. 你听到“文体场馆无形资产开发”，第一眼觉得这是什么？
2. 你会把它归到咨询公司、广告代理、招商服务，还是场馆运营？
3. 你觉得这个词是否足够严肃、可信、适合一号位？
```

After two-sentence explanation:

```text
1. 你是否能理解它不是普通场馆运营咨询？
2. “一号位资产运营工程”是否更能表达高层决策价值？
3. “把沉睡资产变成可持续现金流”是否足够清楚？
4. 哪个表达更适合放在对外方案封面？
```

Budget-pull test:

```text
1. 如果你是场馆一号位，你愿意为什么付费：方案、资源、创收结果、政绩证明，还是长期分润？
2. 从“无形资产收益占比 15% 到 50%-70%”这个目标是否有预算牵引？
3. 哪个词最像一个值得进入年度预算的战略项目？
```

Repeatability test:

```text
3-5 分钟后，你还记得这个服务叫什么吗？
你会怎么向另一个场馆负责人介绍？
```
