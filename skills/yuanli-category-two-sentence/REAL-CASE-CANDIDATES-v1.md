## 1. Scoring Criteria

Each case is scored 1-5 across eight dimensions:

| Dimension | Meaning |
|---|---|
| Category Leap | Old category → new category is clear and powerful |
| Expensive Problem | The pain is important enough to shift budget/attention |
| Two-Sentence Compressibility | Can be compressed into the skill output format |
| Old Category Escape | Escapes an obvious comparison trap |
| Proof Signal | Has concrete business / user / system signal |
| Yuanli Taste | Strategic, elegant, non-generic, high-concept |
| Anonymization Safety | Can be summarized without exposing private details |
| B2 Validation Value | Helps validate B2 品类独创 specifically |

Weighted judgment:

```text
B2 Validation Value + Category Leap + Two-Sentence Compressibility are most important.
```
