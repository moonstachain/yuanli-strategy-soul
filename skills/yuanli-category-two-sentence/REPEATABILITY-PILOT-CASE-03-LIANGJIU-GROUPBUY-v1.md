## 2. 旧品类陷阱

The project can easily be misclassified as:

```text
社群团购
白牌团购
私域卖货
低价电商
微商
拼多多式团购
普通供应链平台
```

Why this is dangerous:

```text
If users classify it as cheap group-buying, they will compare it only by low price and product list.
That misses the real premium logic: certainty, trust, supply-chain efficiency, and reverse-cycle cash-flow network.
```

```text
下沉市场消费者怕买贵、怕买差、怕被割韭菜，需要“闭眼买不吃亏”的确定性。
代理商和实体老板在主业承压时，需要低门槛、可复制、能持续产生现金流的副业系统。
```

## 5. 两句话品类表达

### Recommended version

```text
良久团购帮助下沉市场家庭，把日常消费从反复比价和担心踩坑，升级为轻奢品质、十分之一价格、闭眼买不吃亏的生活刚需系统。
它不同于普通低价团购，因为它不是靠便宜吸引一次购买，而是用源头供应链、熟人信任网络和规模压价飞轮，形成消费者省钱、代理商赚钱的逆周期现金流网络。
```

### More strategic version

```text
良久团购是 K 型社会底盘的极致性价比生活刚需系统。
它解决的不是买便宜货的问题，而是在消费降级周期里，用供应链和信任网络为普通家庭提供确定性好货，也为实体老板提供一张逆周期搞钱门票。
```

### Plain-language control

```text
好货便宜、闭眼买不吃亏的生活团购。
```

## 6. 用户测试问题

Blind-name test:

```text
1. 你听到“下沉市场极致性价比生活刚需系统”，第一眼觉得这是什么？
2. 你会把它归到社群团购、拼多多、微商，还是别的？
3. “逆周期现金流网络”听起来像消费者语言，还是合伙人语言？
```

After two-sentence explanation:

```text
1. 你是否能理解它不是普通低价团购？
2. “轻奢品质，十分之一价格，闭眼买不吃亏”是否比“极致性价比”更容易转述？
3. 对消费者来说，最打动你的是省钱、确定性、信任，还是好货？
4. 对代理商来说，最打动你的是副业、现金流、低门槛，还是团队系统？
```

Budget-pull test:

```text
1. 如果你是消费者，你愿意为什么持续买：便宜、好货、信任、确定性，还是不用再比价？
2. 如果你是实体老板，你是在买货，还是在买逆周期现金流机会？
3. 哪个词最能让你觉得这不是低价卖货？
```

Repeatability test:

```text
3-5 分钟后，你还记得这个业务叫什么吗？
你会怎么向一个家庭主妇介绍？
你会怎么向一个实体老板介绍？
```
